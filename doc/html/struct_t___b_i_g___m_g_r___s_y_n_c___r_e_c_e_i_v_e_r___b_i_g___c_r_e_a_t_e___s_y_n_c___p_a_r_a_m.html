◆ encryption

uint8_t encryption

Identify the encryption mode of the BIG.

  • 0x00: Unencrypted.
  • 0x01: Encrypted.

◆ broadcast_code

uint8_t broadcast_code[16]

Used to generate the session key to decrypt payloads of an encrypted BIS.

◆ mse

uint8_t mse

The maximum number of subevents that a Controller should use to receive data payloads in each interval for a BIS.

  • 0x00: The Controller can schedule reception of any number of subevents up to NSE.
  • 0x01 to 0x1F: Maximum number of subevents that should be used to receive data payloads in each BIS event.

◆ big_sync_timeout

uint16_t big_sync_timeout

Specify the maximum permitted time between successful receptions of BIS PDUs.

  • Range: 0x000A to 0x4000.
  • Time = N*10 ms.
  • Time Range: 100 ms to 163.84 s.
