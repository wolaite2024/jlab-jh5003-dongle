Enumerator
IDU_DECOMPRESS_FINISH_INT 

Triggered when entire decompress procedure is finished.

IDU_LINE_DECOMPRESS_FINISH_INT 

Triggered when decompress procedure of each line is finished.

IDU_RX_FIFO_THRESHOLD_INT 

Triggered when data level is greater than RX FIFO threshold.

IDU_RX_FIFO_OVERFLOW_INT 

Triggered when overflow occurs in RX FIFO.

IDU_TX_FIFO_THRESHOLD_INT 

Triggered when data level is less than TX FIFO threshold.

IDU_TX_FIFO_UNDERFLOW_INT 

Triggered when underflow occurs in TX FIFO.

IDU_DECOMPRESS_ERROR_INT 

Triggered when input data error is detected.
