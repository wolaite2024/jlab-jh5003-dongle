◆ T_LE_CONN_PARAM_TYPE

enum T_LE_CONN_PARAM_TYPE

LE connection parameter types.

Enumerator
GAP_PARAM_CONN_BD_ADDR 

Address of connected device. Read only. Size is uint8[B_MAX_ADV_LEN]. Set to all zeros when not connected.

GAP_PARAM_CONN_BD_ADDR_TYPE 

Address type of connected device. Read only. Size is uint8. Set to zero when not connected.

GAP_PARAM_CONN_INTERVAL 

Current connection interval. Read only. Size is uint16. Range is 7.5ms to 4 seconds (0x0006 to 0x0C80). Default is 0 (no connection).

GAP_PARAM_CONN_LATENCY 

Current slave latency. Read only. Size is uint16. Range is 0 to 499. Default is 0 (no slave latency or no connection).

GAP_PARAM_CONN_TIMEOUT 

Current timeout value. Read only. size is uint16. Range is 100ms to 32 seconds. Default is 0 (no connection).

GAP_PARAM_CONN_MTU_SIZE 

MTU size. Read only. size is uint16.

GAP_PARAM_CONN_LOCAL_BD_TYPE 

Local bluetooth address type. Read only. size is 1 byte.

GAP_PARAM_CONN_RX_PHY_TYPE 

Rx phy type. Read only. size is 1 byte.

GAP_PARAM_CONN_TX_PHY_TYPE 

Tx phy type. Read only. size is 1 byte.

GAP_PARAM_CONN_REMOTE_FEATURES 

Remote supported features. Read only.

GAP_PARAM_CONN_CHANN_ALGORITHM 

LE Channel Selection Algorithm used. Read only.

GAP_PARAM_CONN_HANDLE 

LE link connection handle. Read only.
