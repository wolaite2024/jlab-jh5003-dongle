◆ cscs_set_parameter()

bool cscs_set_parameter ( T_CSCS_PARAM_TYPE  param_type,
uint8_t  len,
void *  p_value 
)

Set a cycling speed and cadence service parameter.

This function can be called with a cycling speed and cadence service parameter type and it will set the cycling speed and cadence service parameter. Cycling speed and cadence service parameters are defined in T_CSCS_PARAM_TYPE. If the "len" field is set to the size of a "uint16_t", the "p_value" field must point to data of type "uint16_t".

Parameters
[in]param_typeCycling speed and cadence service parameter type: T_CSCS_PARAM_TYPE.
[in]lenLength of data to write.
[in]p_valuePointer to data to write. This is dependent on the parameter type and will be cast to the appropriate data type (For example: if the data type of param is uint16_t, p_value will be cast to a pointer of uint16_t).
Returns
Operation result.
Return values
trueOperation success.
falseOperation failure.

Example usage

void test(void)
{
bool ret = cscs_set_parameter(CSCS_PARAM_CTL_PNT_PROG_CLR, 0, NULL);
}

Get a cycling speed and cadence parameter.

This function can be called with a cycling speed and cadence parameter type and it will get a cycling speed and cadence parameter. Cycling speed and cadence service parameters are defined in T_CSCS_PARAM_TYPE.

Parameters
[in]param_typeCycling speed and cadence parameter type: T_CSCS_PARAM_TYPE.
[in,out]p_valuePointer to the location to get the parameter value. This is dependent on the parameter type and will be cast to the appropriate data type (For example: if the data type of param is uint16_t, p_value will be cast to a pointer of uint16_t).
Returns
Operation result.
Return values
trueOperation success.
falseOperation failure.

Example usage

void test(void)
{
bool ret = cscs_get_parameter(CSCS_PARAM_WHEEL_REVOL, &cscs_cul_value);
}
