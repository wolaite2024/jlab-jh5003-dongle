Add Simple Bluetooth LE service client to application.

Parameters
[in]app_cbPointer of APP callback function to handle specific client module data.
[in]link_numInitialize link num.
Returns
Client ID of the specific client module.
Return values
0xffFailed.
otherSuccess.

Example usage

{
client_init(client_num);
simple_ble_client_id = simp_ble_add_client(app_client_callback, APP_MAX_LINKS);
}
