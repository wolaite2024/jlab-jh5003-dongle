◆ GPIO_StructInit()

void GPIO_StructInit ( GPIO_InitTypeDef *  GPIO_InitStruct)

Fills each GPIO_InitStruct member with its default value.

Note
The default settings for the GPIO_InitStruct member are shown in the following table:
GPIO_InitStruct Member Default Value
GPIO_PinBit GPIO_Pin_All
GPIO_Mode GPIO_Mode_IN
GPIO_ITCmd DISABLE
GPIO_ITTrigger GPIO_INT_Trigger_LEVEL
GPIO_ITPolarity GPIO_INT_POLARITY_ACTIVE_LOW
GPIO_ITDebounce GPIO_INT_DEBOUNCE_DISABLE
GPIO_ControlMode GPIO_SOFTWARE_MODE
GPIO_DebounceTime 20
Parameters
[in]GPIO_InitStructPointer to a GPIO_InitTypeDef structure which will be initialized.

Example usage

#define GPIO_Test_Pin GPIO_GetPin(P3_0)
void driver_gpio_init(void)
{
RCC_PeriphClockCmd(APBPeriph_GPIOA, APBPeriph_GPIOA_CLOCK, ENABLE);
GPIO_InitTypeDef GPIO_InitStruct;
GPIO_StructInit(&GPIO_InitStruct);
GPIO_InitStruct.GPIO_PinBit = GPIO_Test_Pin;
GPIO_InitStruct.GPIO_Mode = GPIO_Mode_IN;
GPIO_InitStruct.GPIO_ITCmd = ENABLE;
GPIO_InitStruct.GPIO_ITTrigger = GPIO_INT_Trigger_EDGE;
GPIO_InitStruct.GPIO_ITPolarity = GPIO_INT_POLARITY_ACTIVE_LOW;
GPIO_InitStruct.GPIO_ITDebounce = GPIO_INT_DEBOUNCE_ENABLE;
GPIO_InitStruct.GPIO_DebounceTime = 30;
GPIOx_Init(GPIOA, &GPIO_InitStruct);
}
Parameters
GPIO_InitStruct: pointer to a GPIO_InitTypeDef structure which will be initialized.
Return values
None

void GPIOx_INTConfig ( GPIO_TypeDef *  GPIOx,
uint32_t  GPIO_PinBit,
FunctionalState  NewState 
)

Enable the specified GPIO pin interrupt.

Parameters
[in]GPIOxWhere x can be A or B to select the GPIO peripheral GPIO Declaration.
[in]GPIO_PinBitSpecifies the GPIO pins to be configured, please refer to GPIO Pins Define. This parameter can be one of the following values:
  • GPIO_Pin_x, where x can be 0 ~ 31.
[in]NewStateEnable or disable the specified GPIO pin interrupt. This parameter can be one of the following values:
  • ENABLE: Enable the specified GPIO pin interrupt.
  • DISABLE: Disable the specified GPIO pin interrupt.

Example usage

#define GPIO_Test_Pin GPIO_GetPin(P0_0)
void driver_gpio_init(void)
{
RamVectorTableUpdate(GPIO_A0_VECTORn, gpio_handler);
NVIC_InitTypeDef NVIC_InitStruct;
NVIC_InitStruct.NVIC_IRQChannel = GPIO_IRQ;
NVIC_InitStruct.NVIC_IRQChannelPriority = 3;
NVIC_InitStruct.NVIC_IRQChannelCmd = ENABLE;
NVIC_Init(&NVIC_InitStruct);
GPIOx_MaskINTConfig(GPIOA, GPIO_Test_Pin, DISABLE);
GPIOx_INTConfig(GPIOA, GPIO_Test_Pin, ENABLE);
}

Enable the specified GPIO pin interrupt.

GPIO_Pin_xwhere x can be 0 or 31.

◆ GPIOx_GetINTStatus()

__STATIC_ALWAYS_INLINE ITStatus GPIOx_GetINTStatus ( GPIO_TypeDef *  GPIOx,
uint32_t  GPIO_PinBit 
)

Check whether the GPIO interrupt of the specified pin has occurred or not.

Parameters
[in]GPIOxWhere x can be A or B to select the GPIO peripheral GPIO Declaration.
[in]GPIO_PinBitSpecifies the GPIO pins to be configured, please refer to GPIO Pins Define. This parameter can be one of the following values:
  • GPIO_Pin_x, where x can be 0 ~ 31.
Returns
The new state of GPIO interrupt.
Return values
SETThe interrupt of the specified GPIO pin has occurred.
RESETThe interrupt of the specified GPIO pin has not occurred.

Example usage

#define GPIO_Test_Pin GPIO_GetPin(P0_0)
//GPIO interrupt handler function.
void gpio_handler(void)
{
if (GPIOx_GetINTStatus(GPIOA, GPIO_Test_Pin) == SET)
{
GPIOx_MaskINTConfig(GPIOA, GPIO_Test_Pin, ENABLE);
GPIOx_ClearINTPendingBit(GPIOA, GPIO_Test_Pin);
GPIOx_MaskINTConfig(GPIOA, GPIO_Test_Pin, DISABLE);
}
}
