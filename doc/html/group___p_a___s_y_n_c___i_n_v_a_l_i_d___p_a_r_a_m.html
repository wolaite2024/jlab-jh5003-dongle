Detailed Description

Use macro definitions to indicate that synchronization parameters are invalid in certain scenarios. e.g. sync_handle is invalid before device is synchronized to a periodic advertising train.

Macro Definition Documentation

◆ GAP_INVALID_SYNC_ID

#define GAP_INVALID_SYNC_ID   0xFF

A sync_id value of 0xFF means invalid.

◆ GAP_INVALID_SYNC_HANDLE

#define GAP_INVALID_SYNC_HANDLE   0xFFFF

A sync_handle value of 0xFFFF means invalid.
