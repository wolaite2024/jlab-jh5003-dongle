◆ SPI3WIRE_Init()

void SPI3WIRE_Init ( SPI3WIRE_InitTypeDef *  SPI3WIRE_InitStruct)

Initializes the SPI_3WIRE peripheral according to the specified parameters in the SPI3WIRE_InitStruct.

rtl876x_3wire_spi.h

Parameters
[in]SPI3WIRE_InitStructpointer to a SPI3WIRE_InitTypeDef structure that contains the configuration information for the specified SPI_3WIRE peripheral.
Returns
None.

Example usage

void Driver_SPI3WIRE_Init(void)
{
RCC_PeriphClockCmd(APBPeriph_SPI2W, APBPeriph_SPI2W_CLOCK, ENABLE);
SPI3WIRE_InitTypeDef SPI3WIRE_InitStruct;
SPI3WIRE_StructInit(&SPI3WIRE_InitStruct);
SPI3WIRE_InitStruct.SPI3WIRE_SysClock = 20000000;//20M
SPI3WIRE_InitStruct.SPI3WIRE_Speed = 1000000;
SPI3WIRE_InitStruct.SPI3WIRE_Mode = SPI3WIRE_2WIRE_MODE;
SPI3WIRE_InitStruct.SPI3WIRE_ReadDelay = 0x5;
SPI3WIRE_InitStruct.SPI3WIRE_OutputDelay = SPI3WIRE_OE_DELAY_1T;
SPI3WIRE_InitStruct.SPI3WIRE_ExtMode = SPI3WIRE_NORMAL_MODE;
SPI3WIRE_Init(&SPI3WIRE_InitStruct);
}

FlagStatus SPI3WIRE_GetFlagStatus ( uint32_t  SPI3WIRE_FLAG)

Checks whether the specified 3WIRE_SPI flag is set or not.

rtl876x_3wire_spi.h

Parameters
[in]SPI3WIRE_FLAGspecifies the flag to check. This parameter can be one of the following values:
  • SPI3WIRE_FLAG_BUSY: 3wire spi is busy.
  • SPI3WIRE_FLAG_INT_IND: there is 3wire spi interrupt.
  • SPI3WIRE_FLAG_RESYNC_BUSY: resync is busy.
Returns
The new state of SPI3W Flag status.
Return values
SETThe specified SPI3WIRE Flag status is pending.
RESETThe specified SPI3WIRE Flag status is not pending.

Example usage

void SPI3WIRE_SingleWrite(void)
{
uint32_t timeout = 0;
//Check SPI busy or not
while (SPI3WIRE_GetFlagStatus(SPI3WIRE_FLAG_BUSY) == SET)
{
//add user code here.
}
}
