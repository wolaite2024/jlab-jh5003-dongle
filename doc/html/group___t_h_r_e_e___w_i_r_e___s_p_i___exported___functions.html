◆ SPI3WIRE_GetFlagStatus()

FlagStatus SPI3WIRE_GetFlagStatus ( uint32_t  SPI3WIRE_FLAG)

Checks whether the specified 3WIRE_SPI flag is set or not.

Parameters
SPI3WIRE_FLAGspecifies the flag to check. This parameter can be one of the following values:
  • SPI3WIRE_FLAG_BUSY: 3wire spi is busy.
  • SPI3WIRE_FLAG_INT_IND: there is 3wire spi interrupt.
  • SPI3WIRE_FLAG_RESYNC_BUSY: resync busy or not.
Returns
The new state of SPI3W Flag status.
Return values
SETSPI3WIRE Flag status is pending.
RESETSPI3WIRE Flag status is not pending.
