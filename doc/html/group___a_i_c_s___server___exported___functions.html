)

Set AICS service parameter.

aics_mgr.h

Parameters
[in]srv_instance_idService instance id.
[in]param_typeAICS service parameter type: T_AICS_PARAM_TYPE.
[in]value_lenThe value length of p_value.
  • param_type AICS_PARAM_INPUT_STATE, value_len is size of T_AICS_INPUT_STATE.
  • param_type AICS_PARAM_GAIN_SETTING_PROP, value_len is size of T_AICS_GAIN_SETTING_PROP.
  • param_type AICS_PARAM_INPUT_TYPE, value_len is uint8_t.
  • param_type AICS_PARAM_INPUT_STATUS, value_len is uint8_t.
  • param_type AICS_PARAM_INPUT_DES, value_len is the length of audio input description.
[in]p_valuePointer to the AICS parameter data.
[in]set_change_counterWhether to set change counter.
  • true Set change counter when param_type is AICS_PARAM_INPUT_STATE.
  • false Not set change counter when param_type is AICS_PARAM_INPUT_STATE.
Returns
The result of set AICS service parameter.
Return values
trueSet AICS service parameter is successful.
falseSet AICS service parameter failed.
