enum T_GAP_PA_ADV_STATE

GAP PA periodic advertising state.

Enumerator
PA_ADV_STATE_IDLE 

Idle, no periodic advertising.

PA_ADV_STATE_START_EXT_ADV_STATE_IDLE 

Start periodic advertising when extended advertising of the specified advertising set is disabled. A temporary state, haven't received the result.

PA_ADV_STATE_WAIT_EXT_ADV_STATE_ADVERTISING 

Periodic advertising is not started until the extended advertising of the specified advertising set is enabled.

PA_ADV_STATE_START 

Start periodic advertising when extended advertising of the specified advertising set is enabled. A temporary state, haven't received the result.

PA_ADV_STATE_ADVERTISING 

Periodic advertising.

PA_ADV_STATE_STOP 

Stop periodic advertising. A temporary state, haven't received the result.
