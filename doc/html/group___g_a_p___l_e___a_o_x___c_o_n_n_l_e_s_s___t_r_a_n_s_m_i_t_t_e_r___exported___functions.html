)

Request the Controller to set parameters for the transmission of Constant Tone Extensions in periodic advertising.

NOTE: This function can be called after stack is ready. Explanation: If stack is ready, Application will be notified by message GAP_MSG_LE_DEV_STATE_CHANGE with new_state about gap_init_state which is configured as GAP_INIT_STATE_STACK_READY.

If sending request operation is success, the result of setting parameters related parameters will be returned by app_gap_aox_callback with cb_type GAP_MSG_LE_AOX_CONNLESS_TRANSMITTER_SET_CTE_TRANSMIT_PARAMS.

Parameters
[in]adv_handleUsed to identify an advertising set.
[in]cte_lengthConstant Tone Extension length in 8 Î¼s units. Range: 0x02 to max_cte_length supported by controller returned by le_aox_read_antenna_information.
[in]cte_typeT_GAP_AOX_CONNLESS_TRANSMITTER_CTE_TYPES.
[in]cte_countThe number of Constant Tone Extensions to transmit in each periodic advertising interval. Range: 0x01 to 0x10
[in]switching_pattern_lengthThe number of Antenna IDs in the pattern. Range: 0x02 to max_switching_pattern_length supported by controller returned by le_aox_read_antenna_information.
[in]p_antenna_idsAntenna ID in the pattern.
Returns
Send request operation.
Return values
GAP_CAUSE_SUCCESSSend request operation success.
OthersSend request operation failure.

Example usage

void test(void)

T_GAP_CAUSE cause = le_aox_connless_transmitter_set_cte_transmit_params(adv_handle, cte_length, cte_type, cte_count, switching_pattern_length, p_antenna_ids);
}

T_GAP_CAUSE le_aox_connless_transmitter_set_cte_transmit_enable ( uint8_t  adv_handle,
uint8_t  cte_enable 
)

Request the Controller to enable or disable the use of Constant Tone Extensions in periodic advertising.

NOTE: This function can be called after stack is ready. Explanation: If stack is ready, Application will be notified by message GAP_MSG_LE_DEV_STATE_CHANGE with new_state about gap_init_state which is configured as GAP_INIT_STATE_STACK_READY.

If sending request operation is success, the result of enabling or disabling will be returned by app_gap_aox_callback with cb_type GAP_MSG_LE_AOX_CONNLESS_TRANSMITTER_STATE_CHANGE_INFO.

Parameters
[in]adv_handleIdentifier for the advertising set in which Constant Tone Extension is being enabled or disabled.
[in]cte_enableAoA/AoD Connectionless Transmitter CTE Enable flag.
Returns
Send request operation.
Return values
GAP_CAUSE_SUCCESSSend request operation success.
OthersSend request operation failure.

Example usage

void test(void)
{
T_GAP_CAUSE cause = le_aox_connless_transmitter_set_cte_transmit_enable(adv_handle, cte_enable);
}
