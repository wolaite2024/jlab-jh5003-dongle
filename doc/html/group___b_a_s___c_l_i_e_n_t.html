Detailed Description

BAS client.

Application shall register bas client when initialization through bas_add_client function.

Application can start discovery battery service through bas_start_discovery function.

Application can read battery level characteristic value through bas_read_battery_level function.

Application can config and read the notification flag through bas_set_notify and bas_read_notify function.

Application shall handle callback function registered by bas_add_client.
