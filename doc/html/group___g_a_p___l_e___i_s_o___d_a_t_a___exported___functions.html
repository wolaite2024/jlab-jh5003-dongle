Send isochronous data to Controller.

Note
This function is used for ISOCH_ROLE_INITIATOR, ISOCH_ROLE_ACCEPTOR and BIG_MGR_ROLE_ISOC_BROADCASTER.
Parameters
[in]p_dataPoint to data to be sent.
[in]handleConnection handle of the CIS or BIS.
[in]iso_sdu_lenLength of the SDU to be sent.
[in]ts_flagIndicates whether it contains time_stamp. True: contain time_stamp. False: does not contain time_stamp.
[in]time_stampA time in microseconds. The time_stamp is used when ts_flag is True.
[in]pkt_seq_numThe sequence number of the SDU.
Returns
Send request operation.
Return values
GAP_CAUSE_SUCCESSSend request operation success.
OthersSend request operation failure.

Example usage

void test(void)
{
T_GAP_CAUSE cause = gap_iso_send_data(p_data, handle, iso_sdu_len, ts_flag, time_stamp, pkt_seq_num);
}
