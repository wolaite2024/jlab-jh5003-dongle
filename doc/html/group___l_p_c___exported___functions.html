void LPC_Init ( LPC_InitTypeDef *  LPC_InitStruct)

Initializes LPC peripheral according to the specified parameters in LPC_InitStruct.

rtl876x_lpc.h

Parameters
[in]LPC_InitStructPointer to a LPC_InitTypeDef structure that contains the configuration information for the specified LPC peripheral.
Returns
None.

Example usage

#define LPC_CAPTURE_PIN ADC_0 //ADC_0 as input
#define LPC_COMP_VALUE 0x4
void driver_lpc_init(void)
{
LPC_InitTypeDef LPC_InitStruct;
LPC_StructInit(&LPC_InitStruct);
LPC_InitStruct.LPC_Channel = LPC_CAPTURE_PIN;
LPC_InitStruct.LPC_Edge = LPC_Vin_Over_Vth;
LPC_InitStruct.LPC_Threshold = LPC_400_mV;
LPC_Init(&LPC_InitStruct);
}

void LPC_INTConfig ( uint32_t  LPC_INT,
FunctionalState  NewState 
)

Enables or disables the specified LPC interrupts.

rtl876x_lpc.h

Parameters
[in]LPC_INTSpecifies the LPC interrupt source to be enabled or disabled. This parameter can be one of the following values:
  • LPC_INT_VOLTAGE_COMP: LPC voltage detection interrupt source. If Vin<Vth, cause this interrupt.
  • LPC_INT_COUNT_COMP: LPC couter comparator interrupt source.
[in]NewStateNew state of the specified LPC interrupt. This parameter can be one of the following values:
  • ENABLE: Enable the specified LPC interrupts.
  • DISABLE: Disable the specified LPC interrupts.
Returns
None.

Example usage

#define LPC_CAPTURE_PIN ADC_0 //ADC_0 as input
#define LPC_COMP_VALUE 0x4
void driver_lpc_init(void)
{
LPC_INTConfig(LPC_INT_COUNT_COMP, ENABLE);
RTC_CpuNVICEnable(ENABLE);
RamVectorTableUpdate(RTC_VECTORn, lpc_rtc_handler);
NVIC_InitTypeDef NVIC_InitStruct;
NVIC_InitStruct.NVIC_IRQChannel = RTC_IRQn;
NVIC_InitStruct.NVIC_IRQChannelPriority = 3;
NVIC_InitStruct.NVIC_IRQChannelCmd = ENABLE;
NVIC_Init(&NVIC_InitStruct);
}
Parameters
LPC_INTspecifies the LPC interrupt source to be enabled or disabled. This parameter can be one of the following values:
  • LPC_INT_VOLTAGE_COMP: voltage detection interrupt.If Vin<Vth, cause this interrupt.
  • LPC_INT_COUNT_COMP: couter comparator interrupt.
NewStatenew state of the specified LPC interrupt. This parameter can be: ENABLE or DISABLE.
Return values
None
