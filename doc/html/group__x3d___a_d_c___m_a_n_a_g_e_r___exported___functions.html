◆ adc_mgr_set_int_para()

bool adc_mgr_set_int_para ( uint8_t  index,
uint32_t  enable_intr_bit_map 
)

Set interrupt related parameters.

Note
Make sure the index corresponding channel is valid.
Parameters
indexThe number corresponding to a channel.
enable_intr_bit_mapSpecifies the interrupt pending bit to enable. This parameter can be any combination of the following values:
  • ADC_INT_ONE_SHOT_DONE: ADC one convert end interrupt.
  • ADC_INT_FIFO_OVERFLOW: ADC FIFO overflow interrupt.
  • ADC_INT_FIFO_RD_REQ: ADC read FIFO request interrupt.
  • ADC_INT_FIFO_RD_ERR: ADC read FIFO error interrupt.
  • ADC_INT_FIFO_TH: FIFO larger than threshold.
Returns
Status.
Return values
trueStatus successful.
falseStatus fail.
