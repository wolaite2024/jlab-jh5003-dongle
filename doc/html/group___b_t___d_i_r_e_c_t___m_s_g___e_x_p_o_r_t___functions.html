◆ gap_register_direct_cb()

void gap_register_direct_cb ( P_FUN_BT_DIRECT_CB  app_callback)

Register callback to gap, when messages in Bluetooth Direct Message Callback Type happens, it will callback to APP.

Parameters
[in]app_callbackCallback function provided by the APP to handle Bluetooth direct messages.
  • NULL -> Do not send Bluetooth direct messages to APP.
  • Others -> Use application-defined callback function.
Returns
void.
