Detailed Description

Tx power service.

The Tx Power service uses the Tx Power Level characteristic to expose the current transmit power level of a device when in a connection.The Tx Power service contains only a Tx Power Level characteristic. The Tx Power Service generally makes up a profile with some other services, such as Proximity, and its role is to indicate a device's transmit power level when in a connection.

Application shall register Tx Power service when initialization through tps_add_service function.

Application can set the TX power value through tps_set_parameter function.
