◆ T_GPIO_TYPE

enum T_GPIO_TYPE
Enumerator
GPIO_TYPE_AUTO 

The GPIO function switch automatically, use core GPIO in active mode, use aon GPIO in low power mode.

GPIO_TYPE_CORE 

The GPIO function in core domain, only works when active mode. Support all the features.

GPIO_TYPE_AON 

The GPIO function in aon domain, works in dlps/power down, much slower than CORE mode, not support read level and debounce.

GPIO_TYPE_AUTO 

The GPIO function switch automatically, use core GPIO in active mode, use aon GPIO in low power mode.

GPIO_TYPE_CORE 

The GPIO function in core domain, only works when active mode. Support all the features.

GPIO_TYPE_AON 

The GPIO function in aon domain, works in DLPS/power down, much slower than CORE mode. Reading level and interrupt are not supported for GPIO_TYPE_AON. If gpio type set to GPIO_TYPE_AON in input mode, hal_gpio will switch to GPIO_TYPE_AUTO.
