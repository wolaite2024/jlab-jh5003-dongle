__STATIC_ALWAYS_INLINE void KeyScan_ClearFlags ( KEYSCAN_TypeDef *  KeyScan,
uint32_t  KeyScan_FLAG 
)

Clears KeyScan pending flags.

Parameters
KeyScanselected KeyScan peripheral.
KeyScan_FLAGspecifies the flag to clear. This parameter can be one of the following values:
  • KEYSCAN_FLAG_FIFOLIMIT
  • KEYSCAN_FLAG_DATAFILTER
  • KEYSCAN_FLAG_OVR
Note
  • KEYSCAN_FLAG_FULL and KEYSCAN_FLAG_EMPTY can't be cleared manually. They are cleared by hardware automatically.
