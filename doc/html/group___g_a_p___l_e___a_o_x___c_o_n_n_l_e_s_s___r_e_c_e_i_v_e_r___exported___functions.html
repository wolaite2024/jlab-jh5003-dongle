Request the Controller to enable or disable capturing IQ samples from the Constant Tone Extensions of periodic advertising packets in the periodic advertising train.

If sending request operation is success, the result of enabling or disabling will be returned by app_gap_aox_callback with cb_type GAP_MSG_LE_AOX_CONNLESS_RECEIVER_SET_IQ_SAMPLING_ENABLE.

IQ information from the Constant Tone Extension of a received advertising packet forming part of the periodic advertising train will be reported by app_gap_aox_callback with cb_type GAP_MSG_LE_AOX_CONNLESS_RECEIVER_CONNLESS_IQ_REPORT_INFO.

Parameters
[in]sync_idIdentify the periodic advertising train.
[in]sampling_enableAoA/AoD Connectionless Receiver Sampling Enable flag.
[in]slot_durationsT_GAP_SLOT_DUATIONS_TYPE.
[in]max_sampled_ctesSpecify the maximum number of Constant Tone Extensions in each periodic advertising event that the Controller should collect and report IQ samples from. 0x00: Sample and report all available Constant Tone Extensions 0xXX: The maximum number of Constant Tone Extensions to sample and report in each periodic advertising interval Range: 0x01 to 0x10
[in]switching_pattern_lengthThe number of Antenna IDs in the pattern. Range: 0x02 to max_switching_pattern_length supported by controller returned by le_aox_read_antenna_information.
[in]p_antenna_idsAntenna ID in the pattern.
Returns
Send request operation.
Return values
GAP_CAUSE_SUCCESSSend request operation success.
OthersSend request operation failure.
