bool bt_hfp_ag_audio_connect_req ( uint8_t  bd_addr[6])

Send a request to creat audio connection.

bt_hfp_ag.h

Parameters
[in]bd_addrRemote BT address.
Returns
The status of sending the command request.
Return values
trueCommand request was sent successfully.
falseCommand request was failed to send.
Note
Although the Audio Connection may be triggered by either the AG or the HF, the Codec Connection and the Synchronous Connection shall always be established by the AG (unless one of the devices is a legacy device). AG implementations that comply with the Hands-Free Profile specification Rev. 0.96,1.0 or 1.5, shall not indicate support for the Codec Negotiation feature and shall neither send the +BCS unsolicited response to the HF. In order to guarantee backward compatibility, HFP Rev. x.y implementations shall be able to handle establishment of synchronous connections according to Hands-Free Profile specification Rev. 1.0 or 1.5.
