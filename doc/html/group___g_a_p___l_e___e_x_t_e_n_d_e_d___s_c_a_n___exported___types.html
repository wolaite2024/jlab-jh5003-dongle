enum T_LE_EXT_SCAN_PARAM_TYPE

Extended scan parameter type.

Enumerator
GAP_PARAM_EXT_SCAN_LOCAL_ADDR_TYPE 

The type of address being used in the scan request packets. Read/Write. Size is uint8_t. Default is GAP_LOCAL_ADDR_LE_PUBLIC (T_GAP_LOCAL_ADDR_TYPE).

GAP_PARAM_EXT_SCAN_FILTER_POLICY 

Scanning filter policy. Read/Write. Size is uint8_t. Default is GAP_SCAN_FILTER_ANY (T_GAP_SCAN_FILTER_POLICY).

GAP_PARAM_EXT_SCAN_PHYS 

Scanning PHYs. window. Read/Write. Size is uint8_t. Default is GAP_EXT_SCAN_PHYS_1M_BIT (Extended Scan PHY).

GAP_PARAM_EXT_SCAN_FILTER_DUPLICATES 

Scan Filter Duplicates. Read/Write. Size is uint8_t. Default is GAP_SCAN_FILTER_DUPLICATE_ENABLE (T_GAP_SCAN_FILTER_DUPLICATE).

GAP_PARAM_EXT_SCAN_DURATION 

Scan duration. Read/Write. Size is uint16_t. In units of 10ms, range: 0x0000~0xFFFF. Default is zero.

GAP_PARAM_EXT_SCAN_PERIOD 

Scan period. Read/Write. Size is uint16_t. In units of 1.28sec, range: 0x0000~0xFFFF. Default is zero.
