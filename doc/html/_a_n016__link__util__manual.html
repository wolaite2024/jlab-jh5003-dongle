1 Introduction

The purpose of this document is to give an overview of link util.

  1. the alloc and free function of the BR/EDR link context
    the alloc and free function of the BLE link context

  2. the get function of the BR/EDR link context and status
    the get function of the BLE link context and status

  3. the b2b link judge function
    the b2s link judge function

2 Source Code Overview

This section discribes the main parts of link util source code.

Main API

//alloc the BR/EDR link by bluetooth address
T_APP_BR_LINK *app_link_alloc_br_link(uint8_t *bd_addr);
//free the BR/EDR link
bool app_link_free_br_link(T_APP_BR_LINK *p_link);
-------------------------------------------------------------------
//alloc the BLE link by slot id
T_APP_LE_LINK *app_link_alloc_le_link_by_conn_id(uint8_t conn_id);
//free the BLE link
bool app_link_free_le_link(T_APP_LE_LINK *p_link);
===================================================================
//get all BR/EDR link connected profiles
uint32_t app_link_conn_profiles(void);
//get the b2s link num
uint8_t app_link_get_b2s_link_num(void);
//get BR/EDR link num wich connected with source by the mask profile
uint8_t app_link_get_b2s_link_num_with_profile(uint32_t profile_mask);
//get current SCO link number
uint8_t app_link_get_sco_conn_num(void);
//get current a2dp start number
uint8_t app_link_get_a2dp_start_num(void);
//get the BR/EDR link context by bluetooth address
T_APP_BR_LINK *app_link_find_br_link(uint8_t *bd_addr);
//get the b2s link context by bluetooth address
T_APP_BR_LINK *app_link_find_b2s_link(uint8_t *bd_addr);
-------------------------------------------------------------------
//get the BLE link by connected id
T_APP_LE_LINK *app_link_find_le_link_by_conn_id(uint8_t conn_id);
//get the BLE link by connected handle
T_APP_LE_LINK *app_link_find_le_link_by_conn_handle(uint16_t conn_handle);
//get the BLE link by bluetooth address
T_APP_LE_LINK *app_link_find_le_link_by_addr(uint8_t *bd_addr);
//get the BLE link num
uint8_t app_link_get_le_link_num(void);
//get the BLE encrypted link num
uint8_t app_link_get_le_encrypted_link_num(void);
===================================================================
//judge if the link is b2b link by bluetooth address

bool app_link_check_b2b_link_by_id(uint8_t id);
