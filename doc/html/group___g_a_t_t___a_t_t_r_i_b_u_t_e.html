◆ UUID_128BIT_SIZE

#define UUID_128BIT_SIZE   16

128 bits UUID size.

◆ ATT_ATTRIBUTE_MAX_LENGTH

#define ATT_ATTRIBUTE_MAX_LENGTH   512

Attribute Max length.

◆ HI_WORD

#define HI_WORD (   x)    ((uint8_t)((x & 0xFF00) >> 8))

◆ LO_WORD

#define LO_WORD (   x)    ((uint8_t)(x))

◆ ATTRIB_FLAG_UUID_128BIT

#define ATTRIB_FLAG_UUID_128BIT   0x0001

Attribute uses 128 bit UUID

◆ ATTRIB_FLAG_VALUE_INCL

#define ATTRIB_FLAG_VALUE_INCL   0x0002

Attribute value is included following 16 bit UUID

◆ ATTRIB_FLAG_VALUE_APPL

#define ATTRIB_FLAG_VALUE_APPL   0x0004

Application has to supply write value

◆ ATTRIB_FLAG_ASCII_Z

#define ATTRIB_FLAG_ASCII_Z   0x0008

Attribute value is ASCII_Z string

◆ ATTRIB_FLAG_CCCD_APPL

#define ATTRIB_FLAG_CCCD_APPL   0x0010

Application will be informed about CCCD value is changed

◆ ATTRIB_FLAG_CCCD_NO_FILTER

#define ATTRIB_FLAG_CCCD_NO_FILTER   0x0020

Application will be informed about CCCD value when CCCD is write by client, no matter it is changed or not

◆ ATTRIB_FLAG_INCLUDE_MULTI

#define ATTRIB_FLAG_INCLUDE_MULTI   0x0040

◆ ATTRIB_FLAG_LE

#define ATTRIB_FLAG_LE   0x0800

Used only for primary service declaration attributes if GATT over BLE is supported

◆ ATTRIB_FLAG_BREDR

#define ATTRIB_FLAG_BREDR   0x0400

Used only for primary service declaration attributes if GATT over BR/EDR is supported
