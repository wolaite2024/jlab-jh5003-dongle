◆ app_led_change_mode()

bool app_led_change_mode ( T_LED_MODE  change_mode,
bool  forced_sec_change,
bool  relay_to_sec 
)

A decision-making function set before app_led_set_mode. To make the final decision of LED. If led change mode is not equal to current led mode, app will set led mode again.

Parameters
change_modeLED mode which is expected to change to.
forced_sec_changeIndicated if LED mode is forced to change to change_mode. Non-repeat modes would set to true. If set to false, secondary bud may not change LED mode if its current LED mode is an indepedent mode.
relay_to_secIndicated if this LED mode needs to sync with secondary bud. Only valid for primary bud.
Returns
change result
