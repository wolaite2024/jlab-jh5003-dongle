◆ lns_add_service()

uint8_t lns_add_service ( void *  p_func)

Add location and navigation service to the BLE stack database.

Parameters
[in]p_funcCallback when service attribute was read, write or cccd update.
Returns
Service id generated by the BLE stack: T_SERVER_ID.
Return values
0xFFOperation failure.
othersService id assigned by stack.

Example usage

void profile_init()
{
server_init(1);
lns_id = lns_add_service(app_handle_profile_message);
}

◆ lns_set_parameter()

bool lns_set_parameter ( T_LNS_PARAM_TYPE  param_type,
uint8_t  len,
void *  p_value 
)

Set a Location And Navigation service parameter.

NOTE: You can call this function with a Location And Navigation service parameter type and it will set the Location And Navigation service parameter. Location And Navigation service parameters are defined in T_LNS_PARAM_TYPE. If the "len" field sets to the size of a "uint16_t" ,the "p_value" field must point to a data with type of "uint16_t".

Parameters
[in]param_typeLocation And Navigation service parameter type: T_LNS_PARAM_TYPE
[in]lenLength of data to write
[in]p_valuePointer to data to write. This is dependent on the parameter type and WILL be cast to the appropriate data type (For example: if data type of param is uint16_t, p_value will be cast to pointer of uint16_t).
Returns
Operation result.
Return values
trueOperation success.
falseOperation failure.

Example usage

void test(void)
{
T_LOCATION_AND_SPEED_VALUE_FLAG flag;
flag.instantaneous_speed_present = 1;
flag.total_distance_present = 1;
flag.location_present = 1;
flag.elevation_present = 1;
flag.heading_present = 1;
flag.rolling_time_present = 1;
flag.utc_time_present = 1;
flag.position_status = 1;
flag.speed_and_distance_format = 1;
flag.elevation_source = 1;
flag.heading_source = 1;
flag.rfu = 0;
lns_set_parameter(LNS_LAS_PARAM_INC_FLAG, 2, &flag);
}

Send location and speed value notification data. Application shall call lns_set_parameter to set location and speed value first, and the call this api to send the notication value.

{

flag.instantaneous_speed_present = 1;
flag.total_distance_present = 1;
flag.location_present = 1;
flag.elevation_present = 1;
flag.heading_present = 1;
flag.rolling_time_present = 1;
flag.utc_time_present = 1;
flag.position_status = 1;
flag.speed_and_distance_format = 1;
flag.elevation_source = 1;
flag.heading_source = 1;
flag.rfu = 0;
lns_set_parameter(LNS_LAS_PARAM_INC_FLAG, 2, &flag);
