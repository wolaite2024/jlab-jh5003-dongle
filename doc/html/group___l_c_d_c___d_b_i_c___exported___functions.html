◆ DBIC_Cmd()

void DBIC_Cmd ( FunctionalState  NewState)

Enable or disable DBIC interface.

Parameters
[in]NewStateNew state of DBIC interface. This parameter can be one of the following values:
  • ENABLE: Enable DBIC interface.
  • DISABLE: Disable DBIC interface.

Example usage

void driver_lcdc_dbic_init(void)
{
DBIC_Cmd(ENABLE);
}

◆ DBIC_SwitchMode()

void DBIC_SwitchMode ( uint32_t  mode)

Switch DBIC interface to user mode or auto mode. It can not be changed while SPIC is busy.

Parameters
[in]modeThe mode of DBIC interface. This parameter can be one of the following values:
  • DBIC_AUTO_MODE: Auto mode. User can't push data to FIFO and Enable DBIC interface. SPIC will accept auto read/write command.
  • DBIC_USER_MODE: User mode. User can push data to FIFO and Enable DBIC interface. SPIC won't accept auto read/write command.

Example usage

void driver_lcdc_dbic_init(void)
{
DBIC_SwitchMode(DBIC_USER_MODE);
}

◆ DBIC_SwitchDirect()

void DBIC_SwitchDirect ( uint32_t  dir)

Switch the transfer mode.

Parameters
[in]dirThe transmission mode of DBIC. This parameter can be one of the following values:

  • DBIC_TMODE_RX: Receive mode.

Example usage

void driver_lcdc_dbic_init(void)
{
DBIC_SwitchDirect(DBIC_TMODE_TX);
}

◆ DBIC_StructInit()

void DBIC_StructInit ( LCDC_DBICCfgTypeDef *  DBIC_InitStruct)

Initialize the LCDC_DBICCfgTypeDef structure with default values.

Parameters
[in]DBIC_InitStructPointer to a LCDC_DBICCfgTypeDef structure that will be initialized.
Note
Default values for each member:
Member Default Value Description
DBIC_SPEED_SEL 1 Clock divider value
DBIC_TxThr 0 TX FIFO threshold
DBIC_RxThr 0 RX FIFO threshold
SCPOL DBIC_SCPOL_LOW Serial clock polarity
SCPH DBIC_SCPH_1Edge Serial clock phase

Example usage

void driver_lcdc_dbic_init(void)
{
LCDC_DBICCfgTypeDef dbic_init;
DBIC_StructInit(&dbic_init);
DBIC_Init(&dbic_init);
}

◆ DBIC_Init()

void DBIC_Init ( LCDC_DBICCfgTypeDef *  DBIC_InitStruct)

Initializes DBIC interface according to the specified parameters in the DBICCfg.

Parameters
[in]DBIC_InitStructPointer to a LCDC_DBICCfgTypeDef structure that contains the configuration information for DBIC interface.

Example usage

void driver_lcdc_dbic_init(void)
{
LCDC_DBICCfgTypeDef dbic_init = {0};
dbic_init.DBIC_SPEED_SEL = 2;
dbic_init.DBIC_TxThr = 0;
dbic_init.DBIC_RxThr = 0;
dbic_init.SCPOL = DBIC_SCPOL_LOW;
dbic_init.SCPH = DBIC_SCPH_1Edge;
DBIC_Init(&dbic_init);
}
