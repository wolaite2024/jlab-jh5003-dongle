#define TBS_CALL_CONTROL_POINT_OPTIONAL_OPCODES_CHAR_BIT_LOCAL_HOLD   0

Bit 0: Local Hold.

  • 0: Local Hold and Local Retrieve Call Control Point Opcodes not supported.
  • 1: Local Hold and Local Retrieve Call Control Point Opcodes supported.
