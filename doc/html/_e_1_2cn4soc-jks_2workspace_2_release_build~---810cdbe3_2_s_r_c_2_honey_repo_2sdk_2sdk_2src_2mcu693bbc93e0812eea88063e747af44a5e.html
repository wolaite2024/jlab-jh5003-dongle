Initializes the memory allocation functions used internally by the JPU driver.

Parameters
[in]jmallocPointer to a memory allocation function with the same signature as malloc (i.e., void *jmalloc(size_t size)). This function will be used internally for dynamic memory allocation within the JPU driver.
[in]jfreePointer to a memory free function with the same signature as free (i.e., void jfree(void *ptr)). This function will be used internally to release memory allocated by jmalloc.
Note
This function must be called before using any JPU encoding or decoding APIs that require dynamic memory allocation.
// Example using standard C driver malloc and free:
hal_jpu_mem_init(malloc, free);
