Detailed Description

Use the combination of macro definitions to set extended advertising related parameters for a specified advertising set by calling le_ext_adv_start_setting.

Macro Definition Documentation

◆ EXT_ADV_SET_AUTO

#define EXT_ADV_SET_AUTO   0x00

Automatically set extended advertising related parameters (including advertising parameters, advertising data and scan response data) according to advertising event properties.

◆ EXT_ADV_SET_ADV_PARAS

#define EXT_ADV_SET_ADV_PARAS   0x01

◆ EXT_ADV_SET_ADV_DATA

#define EXT_ADV_SET_ADV_DATA   0x02

◆ EXT_ADV_SET_SCAN_RSP_DATA

#define EXT_ADV_SET_SCAN_RSP_DATA   0x04

◆ EXT_ADV_SET_RANDOM_ADDR
