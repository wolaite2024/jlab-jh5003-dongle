◆ set_io_power_in_lps_mode

void(* set_io_power_in_lps_mode) (bool on)

Set IO power domain ON/OFF in DLPS mode.

Note
IO power domain would change to _LQ, and certain IO(Sleep LED) would possible need more power which is not available under _LQ mode, and this API is used to keep the _HQ power even when under DLPS mode to make these IOs works as expected
Parameters
ontrue to keep power on, false to let power off
modeDLPS mode, LPSMode
Return values
None
