Used to set preferred connection parameters.

Parameters
[in]conn_idConnection ID for this link.
[in]conn_interval_minValue range: 0x0006 - 0x0C80 (7.5ms - 4000ms, 1.25ms/step).
[in]conn_interval_maxValue range: 0x0006 - 0x0C80 (7.5ms - 4000ms, 1.25ms/step).
[in]conn_latencyValue range: 0x0000 - 0x01F3.
[in]supervision_timeoutValue range: 0x000A - 0x0C80 (100ms - 32000ms, 10ms/step).
Returns
Operation result.
Return values
true
  • Current connection parameters have successfully been set into the LE Host.
  • Current connection parameters already meet the conditions, no need to update.
  • Current link state is busy, it may be updated after the link state changes to idle.
falseThere are some errors.
