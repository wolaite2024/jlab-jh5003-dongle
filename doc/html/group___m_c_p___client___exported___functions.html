Read characteristic value.

If sending operation is successful, the result of read characteristic value will be returned by callback registered by ble_audio_cback_register with msg LE_AUDIO_MSG_MCP_CLIENT_READ_RESULT and buf T_MCP_CLIENT_READ_RESULT.

mcp_client.h

Parameters
[in]conn_handleConnection handle
[in]srv_instance_idService instance id
[in]char_uuid
  • MCS_UUID_CHAR_MEDIA_PLAYER_NAME
  • MCS_UUID_CHAR_TRACK_TITLE
  • MCS_UUID_CHAR_TRACK_DURATION
  • MCS_UUID_CHAR_TRACK_POSITION
  • MCS_UUID_CHAR_MEDIA_STATE
  • MCS_UUID_CHAR_CONTENT_CONTROL_ID
  • MCS_UUID_CHAR_PLAYING_ORDERS_SUPPORTED
  • MCS_UUID_CHAR_MEDIA_CONTROL_POINT_OPCODES_SUPPORTED
[in]gmcs
  • true : Generic Media Control Service (GMCS)
  • false : Media Control Service (MCS)
Returns
The result of sending operation
Return values
trueSending operation is successful
falseSending operation is failed

Write Media Control Point characteristic.

If sending operation is successful and Media Control Server sends Media Control Point Notification, Media Control Point Notification will be returned by callback registered by ble_audio_cback_register with msg LE_AUDIO_MSG_MCP_CLIENT_MEDIA_CP_NOTIFY and buf T_MCP_CLIENT_MEDIA_CP_NOTIFY.
