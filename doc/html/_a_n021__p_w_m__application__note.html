Introduction

The purpose of this document is to give an overview of PWM function.

Source Code Overview

This section discribes the main parts of PWM source code.

Initialization

Configure the specified PWM according to the specified parameters in the T_PWM_CONFIG structure.

bool pwm_config(T_PWM_HANDLE handle, T_PWM_CONFIG *config_struct);

Configure PWM output pin.

bool pwm_pin_config(T_PWM_HANDLE handle, uint8_t pin_index, T_PWM_PIN_FUNC func);

Create the PWM timeout callback function.

bool pwm_register_timeout_callback(T_PWM_HANDLE handle, void (*p_timer_callback)(T_PWM_HANDLE));

Change PWM freq and duty according high_count and low_count

bool pwm_change_duty_and_frequency(T_PWM_HANDLE handle, uint32_t high_count, uint32_t low_count);

PWM create/delete/start/stop API

APP will use those following API to create/delete/start/stop PWM.

T_PWM_HANDLE pwm_create(const char *name,
uint32_t high_period_us, uint32_t low_period_us, bool deadzone_supported)
{
T_HW_TIMER *p_hw_timer = (T_HW_TIMER *)hw_timer_pool_alloc(deadzone_supported);
if (p_hw_timer != NULL)
{
p_hw_timer->user_name = name;
p_hw_timer->cback = NULL;
hw_timer_driver_init_pwm_int((T_HW_TIMER_ID)(p_hw_timer->timer_id), high_period_us, low_period_us);
}
return (T_PWM_HANDLE)p_hw_timer;
}
bool pwm_delete(T_PWM_HANDLE handle)
{
return hw_timer_delete(handle);
}
RAM_TEXT_SECTION bool pwm_start(T_PWM_HANDLE handle)
{
return hw_timer_start(handle);
}
RAM_TEXT_SECTION bool pwm_stop(T_PWM_HANDLE handle)
{
return hw_timer_stop(handle);
}

PWM check API

APP will use this following API to check whether the specified PWM has been enabled or disabled.

bool pwm_is_active(T_PWM_HANDLE handle, bool *state);
