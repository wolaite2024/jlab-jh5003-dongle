Macro Definition Documentation

◆ GATT_CHAR_PROP_BROADCAST

#define GATT_CHAR_PROP_BROADCAST   0x01

If set, permits broadcasts of the Characteristic Value using Server Characteristic Configuration Descriptor.

◆ GATT_CHAR_PROP_READ

#define GATT_CHAR_PROP_READ   0x02

If set, permits reads of the Characteristic Value.

◆ GATT_CHAR_PROP_WRITE_NO_RSP

#define GATT_CHAR_PROP_WRITE_NO_RSP   0x04

If set, permits writes of the Characteristic Value without response.

◆ GATT_CHAR_PROP_WRITE

#define GATT_CHAR_PROP_WRITE   0x08

If set, permits writes of the Characteristic Value with response.

◆ GATT_CHAR_PROP_NOTIFY

#define GATT_CHAR_PROP_NOTIFY   0x10

If set, permits notifications of a Characteristic Value without acknowledgment.

◆ GATT_CHAR_PROP_INDICATE

#define GATT_CHAR_PROP_INDICATE   0x20

If set, permits indications of a Characteristic Value with acknowledgment.

◆ GATT_CHAR_PROP_WRITE_AUTHEN_SIGNED

#define GATT_CHAR_PROP_WRITE_AUTHEN_SIGNED   0x40

If set, permits signed writes to the Characteristic Value.

◆ GATT_CHAR_PROP_EXT_PROP

#define GATT_CHAR_PROP_EXT_PROP   0x80

If set, additional characteristic properties are defined in the Characteristic Extended Properties Descriptor.

◆ GATT_CLIENT_CHAR_CONFIG_DEFAULT
