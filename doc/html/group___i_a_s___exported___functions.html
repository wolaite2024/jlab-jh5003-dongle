T_SERVER_ID ias_add_service ( void *  p_func)

Add immediate alert service to the BLE stack database.

Parameters
[in]p_funcCallback when service attribute was read, write or cccd update.
Returns
Service id generated by the BLE stack: T_SERVER_ID.
Return values
0xFFOperation failure.
othersService id assigned by stack.

Example usage

void profile_init()
{
server_init(1);
ias_id = ias_add_service(app_handle_profile_message);
}
