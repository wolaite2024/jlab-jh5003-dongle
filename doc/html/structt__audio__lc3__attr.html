Field Documentation

◆ sample_rate

uint32_t sample_rate

Sample rate for encoding or decoding. The supported values are 8000, 16000, 24000, 32000, 44100 and 48000 in HZ.

◆ chann_location

uint32_t chann_location

Channel location for encoding or decoding. The supported values are 4-octet bitmap defined in Audio Channel Location Bitmap.

◆ frame_length

uint16_t frame_length

Frame length per channel in octets for encoding or decoding. The supported values are from 20 to 400.

◆ frame_duration

T_AUDIO_LC3_FRAME_DURATION frame_duration

Frame duration for encoding. The supported frame durations are defined in T_AUDIO_LC3_FRAME_DURATION.

◆ presentation_delay

uint32_t presentation_delay

Presentation delay for encoding or decoding. The supported values are from 0 to 0xFFFFFFFF in microsecond units.
