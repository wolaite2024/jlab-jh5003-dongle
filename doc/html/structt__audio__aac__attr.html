Field Documentation

◆ sample_rate

uint32_t sample_rate

Sample rate for encoding or decoding. The supported values are 8000, 11025, 12000, 16000, 22050, 24000, 32000, 44100, 48000, 64000, 88200 and 96000 in HZ.

◆ chann_num

uint8_t chann_num

Number of channels for encoding or decoding. The supported numbers are 1 (Mono) and 2 (Stereo).

◆ transport_format

T_AUDIO_AAC_TRANSPORT_FORMAT transport_format

Transport formats for encoding or decoding. The supported values are defined in T_AUDIO_AAC_TRANSPORT_FORMAT.

◆ object_type

T_AUDIO_AAC_OBJECT_TYPE object_type

Object type for encoding or decoding. The supported values are defined in T_AUDIO_AAC_OBJECT_TYPE.

◆ vbr

uint8_t vbr

Variable bitrate (VBR) or constant bitrate (CBR) for encoding. The supported values are 0 for CBR and 1 for VBR.

◆ bitrate

uint32_t bitrate

Bitrate for encoding. The supported ranges are from 0 bit/s to 0x7FFFFF bit/s.

Note
A value of 0 indicates that the bitrate is not known.
