◆ le_register_aox_cb()

void le_register_aox_cb ( P_FUN_LE_AOX_CB  aox_callback)

Register callback to GAP. When messages in GAP LE AoA/AoD Callback Msg happen, it will callback to the application.

Parameters
[in]aox_callbackCallback function provided by the APP to handle GAP aox command messages sent from the GAP.
  • NULL -> Not send GAP aox command messages to APP.
  • Other -> Use application-defined callback function.
Returns
void.

Example usage

void app_le_gap_init(void)
{
......
le_register_aox_cb(app_gap_aox_callback);
}
T_APP_RESULT app_gap_aox_callback(uint8_t cb_type, void *p_cb_data)
{
T_APP_RESULT result = APP_RESULT_SUCCESS;
T_LE_AOX_CB_DATA *p_data = (T_LE_AOX_CB_DATA *)p_cb_data;
switch (cb_type)
{
case GAP_MSG_LE_AOX_READ_ANTENNA_INFORMATION:
APP_PRINT_INFO5("GAP_MSG_LE_AOX_READ_ANTENNA_INFORMATION: cause 0x%x, supported_switching_sampling_rates 0x%x, num_antennae %d, max_switching_pattern_length %d, max_cte_length %d",
p_data->p_le_aox_read_antenna_information_rsp->cause,
p_data->p_le_aox_read_antenna_information_rsp->supported_switching_sampling_rates,
p_data->p_le_aox_read_antenna_information_rsp->num_antennae,
p_data->p_le_aox_read_antenna_information_rsp->max_switching_pattern_length,
p_data->p_le_aox_read_antenna_information_rsp->max_cte_length
);
break;

◆ le_aox_read_antenna_information()

T_GAP_CAUSE le_aox_read_antenna_information ( void  )

Read antenna information. Read the supported_switching_sampling_rates, num_antennae, max_switching_pattern_length, max_cte_length of a transmitted Constant Tone Extension supported by the Controller. Antenna information will be returned by app_gap_aox_callback with cb_type GAP_MSG_LE_AOX_READ_ANTENNA_INFORMATION.

Applications can only call this API after Bluetooth Host is ready.
Explanation: If Bluetooth Host is ready, the application will be notified by message GAP_MSG_LE_DEV_STATE_CHANGE with new_state about gap_init_state which is configured as GAP_INIT_STATE_STACK_READY.

Returns
Result of sending request.
Return values
GAP_CAUSE_SUCCESSSend request success.
GAP_CAUSE_SEND_REQ_FAILEDSend request sent fail.
