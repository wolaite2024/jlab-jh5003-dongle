◆ gls_set_parameter()

bool gls_set_parameter ( T_GLS_PARAM_TYPE  param_type,
uint8_t  len,
void *  p_value 
)

Set a GLS parameter.

This function can be called with a Glucose Service parameter type and it will set the Glucose Service parameter. Glucose Service parameters are defined in T_GLS_PARAM_TYPE. If the "len" field is set to the size of a "uint16_t", the "p_value" field must point to a data with type of "uint16_t".

Parameters
[in]param_typeGlucose Service parameter type: T_GLS_PARAM_TYPE.
[in]lenLength of data to write.
[in]p_valuePointer to data to write. This is dependent on the parameter type and WILL be cast to the appropriate data type (For example: if data type of param is uint16_t, p_value will be cast to pointer of uint16_t).
Returns
Operation result.
Return values
trueOperation success.
falseOperation failure.

Example usage

void test(void)
{
T_GLC_MEASUREMENT_FLAG ms_flag =
{
1, //time_offset
1, //con_ts_loc
GLC_FLAGS_UNITS_MOL_L_ON, //con_units
1, //ss_annuciation
0, //ctxt_info_follows
0 //rfu
};
bool ret = gls_set_parameter(GLS_PARAM_GLC_MS_FLAG, 1, &ms_flag);
}

bool gls_get_parameter ( T_GLS_PARAM_TYPE  param_type,
uint8_t *  len,
void *  p_value 
)

Get a GLS parameter.

This function can be called with a glucose parameter type and it will get a glucose parameter. Glucose parameters are defined in T_GLS_PARAM_TYPE.

Parameters
[in]param_typeGlucose parameter type: T_GLS_PARAM_TYPE
[in,out]lenPointer to the location to get the length of data.
[in,out]p_valuePointer to the location to get the parameter value. This is dependent on the parameter type and will be cast to the appropriate data type (For example: if data type of param is uint16_t, p_value will be cast to pointer of uint16_t).
Returns
Operation result.
Return values
trueOperation success.
falseOperation failure.

Example usage

void test(void)
{
bool ret = gls_get_parameter(GLS_PARAM_RECORD_NUM, &len, &record_num);
}
