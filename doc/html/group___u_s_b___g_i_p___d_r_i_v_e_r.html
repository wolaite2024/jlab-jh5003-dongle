Detailed Description

The module mainly supply components to realize USB GIP Class.
The driver support multi interfaces, such as data interface and audio interface,
The data interface completes the command interaction with the PC, and audio transmits.

How to realize USB GIP interface.

Firstly, allocate a function instance by usb_gip_driver_inst_alloc.

Example
void *demo_instance1 = usb_gip_driver_inst_alloc(); // data interface
void *demo_instance2 = usb_gip_driver_inst_alloc(); // audio interface

Then, realize gip interfaces as follows:

  • Microsoft Specific Descriptors.
    • step1: realize descriptors.
    • step2: register to the USB GIP driver.
      Example – USB GIP
      T_GIP_MS_OS_STRING_DESC demo_ms_os_string_desc =
      {
      ...init...
      };
      T_GIP_MS_EXTD_CMP_ID_DESC demo_ms_extended_compatible_id_desc =
      {
      ...init...
      };
      usb_gip_driver_ms_desc_register((void *) &demo_ms_os_string_desc,
      (void *) &demo_ms_extended_compatible_id_desc);
  • GIP Data Interfaces.
    • step1: realize descriptors array.
    • step2: place gip requests in queue to ensure that each request can be processed.
      Example – USB GIP
      T_USB_INTERFACE_DESC demo_data_std_if_desc =
      {
      ...init...
      };
      T_USB_ENDPOINT_DESC demo_data_int_out_ep_desc_fs =
      {
      ...init...
      };
      T_USB_ENDPOINT_DESC demo_data_int_out_ep_desc_hs =
      {
      ...init...
      };
      T_USB_ENDPOINT_DESC demo_data_int_in_ep_desc_fs =
      {
      ...init...
      };
      T_USB_ENDPOINT_DESC demo_data_int_in_ep_desc_hs =
      {
      ...init...
      };
      void *demo_if_descs_fs[] =
      {
      (void *) &demo_std_if_desc,
      (void *) &demo_data_int_out_ep_desc_fs,
      (void *) &demo_data_int_in_ep_desc_fs,
      NULL,
      };
      void *demo_if_descs_hs[] =
      {
      ...
      NULL,
      };
      void *demo_data_pipe_open(uint8_t ep_addr, T_USB_GIP_ATTR attr, uint8_t pending_req_num,
      USB_GIP_DATA_PIPE_CB cb)
      {
      usb_gip_driver_data_pipe_open(ep_addr, attr, pending_req_num);
      ...
      }
      bool demo_data_pipe_send(void *handle, void *buf, uint32_t len)
      {
      ...
      usb_gip_driver_data_pipe_send();
      ...
      }
      bool demo_data_pipe_close(uint8_t ep_addr)
      {
      ...
      usb_gip_driver_data_pipe_close();
      ...
      }
      usb_gip_driver_if_desc_register(demo_instance1, (void *)demo_if_descs_hs, (void *)demo_if_descs_fs);

GIP Aduio interface

  • step 1: realize interface descriptors array.
  • step 2: initialize streaming attribute.
  • step 3: register cb to process audio data.
    Example – USB GIP
    #include "usb_spec20.h"
    #include "usb_gip_desc.h"
    T_USB_INTERFACE_DESC demo_audio_if_alt0_desc =
    {
    ...init...
    };
    T_USB_INTERFACE_DESC demo_audio_if_alt1_desc =
    {
    ...init...
    };
    T_UAC1_STD_EP_DESC demo_std_iso_in_ep_desc =
    {
    ...init...
    };
    T_UAC1_ISO_EP_DESC demo_cs_iso_out_ep_desc =
    {
    ...init...
    };
    T_USB_DESC_HDR* demo_audio_descs_spk_fs[] =
    {
    (T_USB_DESC_HDR*)&demo_audio_if_alt0_desc,
    (T_USB_DESC_HDR*)&demo_audio_if_alt1_desc,
    (T_USB_DESC_HDR*)&demo_std_iso_out_ep_desc,
    (T_USB_DESC_HDR*)&demo_cs_iso_out_ep_desc,
    NULL,
    };
    ...
    T_USB_DESC_HDR* demo_audio_descs_spk_hs[] =
    {
    ...
    NULL,
    };
    usb_gip_driver_if_desc_register(demo_instance2, demo_audio_descs_spk_fs, demo_audio_descs_spk_hs);
    int demo_usb_audio_upstream(uint8_t *buf, uint16_t len)
    {
    //Process data in buf which length is len
    }
    int demo_usb_audio_downstream(uint8_t *buf, uint16_t len)
    {
    //Process data out buf which length is len
    }
    T_USB_GIP_DRIVER_XMIT_CBS demo_cbs =
    {
    .upstream = demo_usb_audio_upstream,
    .downstream = demo_usb_audio_downstream,
    }
    usb_audio_driver_cb_register(demo_instance, &demo_cbs, proc_interval_out, proc_interval_in);
    int demo_usb_audio_activate(uint8_t dir, uint8_t bit_res, uint32_t sample_rate, uint8_t chan_num)
    {
    //Process
    }
    int demo_usb_audio_deactivate(uint8_t dir)
    {
    //Process
    }
    int32_t demo_vol_attr_set_spk(int value)
    {
    // Process
    }
    int32_t demo_vol_attr_get_spk(int value)
    {
    // Process
    }
    T_USB_GIP_AUDIO_CBS demo_audio_cbs =
    {
    .activate = demo_usb_audio_activate;
    .deactivate = demo_usb_audio_deactivate;
    .volset = demo_vol_attr_set_spk;
    .micset = demo_vol_attr_get_spk;
    }
    GipDeviceCallbackRegister(&demo_audio_cbs);
    Lastly, call usb_gip_driver_init to initialize usb gip driver.

If GIP_DRIVER_CONGESTION_CTRL_DROP_CUR is set, current data to send will be dropped, else the first data in the queue will be dropped.

Note
Only effective for in endpoint.
