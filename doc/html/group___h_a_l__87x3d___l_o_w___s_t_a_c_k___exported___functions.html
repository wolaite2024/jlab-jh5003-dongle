uint8_t(* rws_read_bt_piconet_clk) (uint16_t conn_handle, uint32_t *bb_clk_timer, uint16_t *bb_clk_us)

Read bluetooth piconet clock for application.

Parameters
conn_handleidentify which connection to be used
bb_clk_timerpiconet clk[27:0] (unit : half slot)
bb_clk_uscount up timer (range 0~624 us)
if bb_clk_timer[0] = 1, 312 <= bb_clk_us <= 624;
if bb_clk_timer[0] = 0, 0 <= bb_clk_us <= 312;
Returns
A 8bit value
