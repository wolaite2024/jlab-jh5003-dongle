Relay an asynchronous message to the remote side.

remote.h

Parameters
[in]handleThe remote control relay handle T_REMOTE_RELAY_HANDLE.
[in]msg_idThe asynchronous message id.
[in]msg_bufThe asynchronous message payload buffer.
[in]msg_lenThe asynchronous message payload length.
[in]loopbackSet the asynchronous message loopback or not.
  • true The message is relayed to the remote side and looped back to the sender.

Return values

falseAsynchronous message was failed to relay.

T_REMOTE_TIMER_TYPE  timer_type,

Relay a synchronous message to the remote side.

remote.h

Parameters
[in]handleThe remote control relay handle T_REMOTE_RELAY_HANDLE.
[in]msg_idThe synchronous message id.
[in]msg_bufThe synchronous message payload buffer.
[in]msg_lenThe synchronous message payload length.
[in]timer_typeThe synchronous timer type T_REMOTE_TIMER_TYPE.
[in]timer_periodThe synchronous timer period in milliseconds.
[in]loopbackSet the synchronous message loopback or not.
  • true The message is relayed to the remote side and looped back to the sender.
  • false The message is relayed to the remote side directly.
Returns
The status of relaying the synchronous message.
Return values
trueSynchronous message was relayed successfully.
falseSynchronous message was failed to relay.
Note
If timer_period is 0, remote control will use default period value internally.
