diff the time with high precision as much as possible

The precision is 1us in the range of 0 ~ 1.6 second, otherwise 10ms with the range of 0 ~ 24 day.

Parameters
[in]begin_msthe ms-level begin time
[in]begin_usthe us-level begin time
[in]end_msthe ms-level end time
[in]end_usthe us-level end time
Returns
time elapsed (bit31 indicates the unit of return value: 1 - us, 0 - ms.)
