Detailed Description

The module mainly supply components to realize USB CDC Class.
This driver support virtual port com to communicate with computer.

How to realize USB CDC interface.

Firstly, allocate two function instances by usb_cdc_driver_inst_alloc.

Example
void *demo_instance0 = usb_cdc_driver_inst_alloc();
void *demo_instance1 = usb_cdc_driver_inst_alloc();

Then, realize cdc interfaces as follows:

  • CDC Interfaces.
    • step1: realize descriptors array.
    • step2: place cdc requests in queue to ensure that each request can be processed.
      Example – USB CDC

◆ CDC_DRIVER_CONGESTION_CTRL_DROP_CUR

#define CDC_DRIVER_CONGESTION_CTRL_DROP_CUR   USB_PIPE_CONGESTION_CTRL_DROP_CUR

congestion control

usb_cdc_driver.h

If CDC_DRIVER_CONGESTION_CTRL_DROP_CUR is set, current data to send will be dropped, else the first data in the queue will be dropped.

Note
Only effective for in endpoint.
