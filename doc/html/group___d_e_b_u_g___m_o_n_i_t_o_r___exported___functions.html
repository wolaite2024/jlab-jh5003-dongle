◆ watch_point_0_setting()

static void watch_point_0_setting ( uint32_t  watch_address,
uint32_t  watch_size,
uint32_t  read_write_func 
)
inlinestatic

Configure Watchpoint 0.

Parameters
[in]watch_addressthe address of variable which is being watched
[in]watch_sizethe size of variable which is being watched. the valid value are listed below:
  • DWT_DATAVSIZE_BYTE: Detect 1 byte access
  • DWT_DATAVSIZE_HALF_WORD: Detect 2 bytes access
  • DWT_DATAVSIZE__WORD: Detect 4 bytes access
[in]read_write_functhe detecting type of Watchpoint. the valid value are listed below:
  • DWT_FUNCTION_READ: Detect Read access
  • DWT_FUNCTION_WRITE: Detect Write access
  • DWT_FUNCTION_READ_OR_WRITE: Detect Read or Write access
Returns
none
