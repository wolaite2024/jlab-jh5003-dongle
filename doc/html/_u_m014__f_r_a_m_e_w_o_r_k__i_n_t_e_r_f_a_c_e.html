Introduction

Framework is composed of System Manager, BT Manager, Audio Manager, and RemoteController Manager. It provides a rich API set for customers to develop Audio and Bluetooth products.

System Manager

System Manager(SYSM) is the main component for the interoperability between application and framework. Application shall use API sys_mgr_init() to initialize System Manager. The parameter evt_handle is used for Framework sending messages to application. Below is an example code for framework initialization.

static void framework_init(void)
{
/* System Manager */
sys_mgr_init(audio_evt_queue_handle);
/* Bluetooth Manager */
bt_mgr_init(app_cfg_const.supported_profile_mask);
/* Audio Manager */
audio_mgr_init();
/* RemoteController Manager */
remote_mgr_init((T_REMOTE_SESSION_ROLE)app_cfg_nv.bud_role);
remote_local_addr_set(app_cfg_nv.bud_local_addr);
remote_peer_addr_set(app_cfg_nv.bud_peer_addr);
}

Each application component can register SYSM callback into Framework with API sys_mgr_cback_register(). If Any SYSM EVENT T_SYS_EVENT occurs, Framework will notify every SYSM callback registered by application.

Check System Manager for more detailed API description.

BT Manager

BT Manager(BTM) provides more convenient and high-level APIs for Bluetooth manipulation. You are encouraged to use BT Manager APIs, rather than using GAP & Profile APIs directly. BT Manager divides APIs into two subsets:

  • BT link and device manager APIs
  • BT Profile manager APIs

Check BT Manager for more detailed API description.

Audio Manager

Audio Manager(AM) provides Audio manipulation APIs. Audio volume control APIs are defined in Audio. Ringtone control APIs are defined in Ringtone. Voice prompt control APIs are defined in Voice Prompt. TTS control APIs are defined in Text-to-Speech. EQ effects control APIs are defined in Equalizer.

Check Audio Manager for more detailed API description.

RemoteController Manager

RemoteController Manager(RCM) provides interfaces for application peer-2-peer between identical local and remote side. This module focuses on p2p session setup, p2p information exchange, and p2p roleswap.

Check Remote Control for more detailed API description.
