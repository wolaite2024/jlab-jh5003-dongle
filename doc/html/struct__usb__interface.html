Detailed Description

Core structure to realize interface.

Parameters
if_numinterface number, NOTE: the value may be changed in usb_composite_dev_interface_add
descs_fsfull speed interface descriptors
descs_hshigh speed interface descriptors
epsendpoint list belongs to the interface
ctrl_request_proccallback to process class specific requests
alt_getcallback to process GET_ALT request
alt_setcallback to process SET_ALT request
createcallback to initialize interface, that will be called in usb_composite_dev_interface_add.
And if_num in the input parameter interface may be changed, the interface MUST change bInterfaceNumber
in the interface descriptor to the actual interface number.
releasecallback to release interface related resource
privprivate data
