uint16_t  past_timeout_10ms 
)

BASS set PA sync parameters.

bass_mgr.h

This API is used when receive LE_AUDIO_MSG_BASS_GET_PA_SYNC_PARAM.

Parameters
[in]source_idSource id.
[in]pa_sync_optionsPA create sync options.
[in]pa_sync_skipThe maximum number of periodic advertising events that can be skipped after a is successful receive. Range: 0x0000 to 0x01F3.
[in]pa_sync_timeout_10msPA sync timeout. Range: 0x000A to 0x4000 Time = N*10 ms Time Range: 100 ms to 163.84 s
[in]past_timeout_10msPAST timeout.
Returns
The result of set PA sync parameters.
Return values
trueSet PA sync parameters is successful.
falseSet PA sync parameters failed.
