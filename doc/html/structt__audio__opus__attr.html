Detailed Description

Define Audio OPUS format attributes for AUDIO_FORMAT_TYPE_OPUS.

Field Documentation

◆ sample_rate

uint32_t sample_rate

Sample rate for encoding or decoding. The supported values are 8000, 12000, 16000, 24000 and 48000 in HZ.

◆ chann_num

uint8_t chann_num

Number of channels for encoding or decoding. The supported numbers are 1 (Mono) and 2 (Stereo).

◆ cbr

uint8_t cbr

Constant bitrate (CBR) or variable bitrate (VBR) for encoding. The supported values are 0 for VBR and 1 for CBR.

◆ cvbr

uint8_t cvbr

Constrained variable bitrate (CVBR) or unconstrained variable bitrate (VBR) for encoding. The supported values are 0 for VBR and 1 for CVBR.

Note
CVBR shall be enabled only when CBR disabled.

◆ mode

uint8_t mode

Operating mode for encoding or decoding. The supported modes are 1 for SILK-only, 2 for Hybrid (SILK+CELT), and 3 for CELT-only.

◆ complexity

uint8_t complexity

Computational complexity for encoding. The supported values are from 0 to 10, where 0 is the lowest complexity and 10 is the highest complexity.

◆ frame_duration

T_AUDIO_OPUS_FRAME_DURATION frame_duration

Frame duration for encoding. The supported frame durations are defined in T_AUDIO_OPUS_FRAME_DURATION.

Note
Select 20ms frame duration by default.

◆ bitrate

uint32_t bitrate

Bitrate for encoding. The supported ranges are from 6000 bit/s to 510000 bit/s.
