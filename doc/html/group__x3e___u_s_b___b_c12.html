◆ usb_bc12_type_get()

uint8_t usb_bc12_type_get ( void  )

get USB bc12 type

usb_bc12.h

the bc12 type is defined in T_USB_BC12_TYPE. If the type is BC12_TYPE_OTHERS_USER_DEFINED_0P5A, max charger current is user-defined, and the proposed value is less-than-0.5A for safety. If the type is BC12_TYPE_ADP_ERROR, please DO NOT charge the battery, and call usb_bc12_type_get to get the bc12 type once again.

Returns
bc12 type defined in T_USB_BC12_TYPE
