Detailed Description

HID driver report descriptor parser used in USB_HID_DRIVER_REPORT_DESC_PARSE_CB type: usage type start: report desc start of a HID device start: report desc end of a HID device tlc_start: next byte of tlc, report id can be inserted into this byte include_physical: record if report map include physical field report_ids refer to T_USB_HID_DRIVER_REPORT_IDS.
