◆ vcs_write_cp_by_group()

T_LE_AUDIO_CAUSE vcs_write_cp_by_group ( T_BLE_AUDIO_GROUP_HANDLE  group_handle,
T_VCS_CP_OP  op,
T_VCS_VOLUME_CP_PARAM *  p_param 
)

VCS client sends Volume Control Point by LE audio group.

vcs_client.h

If this API is called successfully, the Application will receive LE_AUDIO_MSG_VCS_CLIENT_CP_RESULT. The parameter op can't be set to VCS_CP_SET_ABSOLUTE_VOLUME, VCS_CP_UNMUTE, and VCS_CP_MUTE. Application can call cap_vcs_change_mute and cap_vcs_change_volume instead.

Experimental Added Since 2.13.0.0:
Parameters
[in]group_handleLE audio group: T_BLE_AUDIO_GROUP_HANDLE.
[in]opVolume control point opcode: T_VCS_CP_OP.
[in]p_paramPointer to Volume Control Point parameter: T_VCS_VOLUME_CP_PARAM.
Returns
The result of VCS client sending Volume Control Point by LE audio group.
Return values
trueVCS client sending Volume Control Point by LE audio group is successful.
falseVCS client sending Volume Control Point by LE audio group failed.
