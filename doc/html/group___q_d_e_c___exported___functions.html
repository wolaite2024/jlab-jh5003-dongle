Example usage

{

RCC_PeriphClockCmd(APBPeriph_QDEC, APBPeriph_QDEC_CLOCK, ENABLE);

QDEC_StructInit(&QDEC_InitStruct);
QDEC_InitStruct.axisConfigY = ENABLE;
QDEC_InitStruct.debounceEnableY = Debounce_Enable;

}

}

uint32_t  QDEC_AXIS 
)

Get Qdecoder Axis(x/y/z) direction.

rtl876x_qdec.h

Parameters
[in]QDECxSelected Qdecoder peripheral, which can be QDEC.
[in]QDEC_AXISSpecifies the Qdecoder axis. This parameter parameter can be one of the following values:
  • QDEC_AXIS_X: The qdecoder X axis.
  • QDEC_AXIS_Y: The qdecoder Y axis.
  • QDEC_AXIS_Z: The qdecoder Z axis.
Returns
The direction of the Qdecoder axis.
Return values
QDEC_AXIS_DIR_UPThe axis is rolling up.
QDEC_AXIS_DIR_DOWNThe axis is rolling down.

Example usage

int16_t y_axis;
uint16_t dir;
void QDEC_Handler(void)
{
if (QDEC_GetFlagState(QDEC, QDEC_FLAG_NEW_CT_STATUS_Y) == SET)
{
//Mask qdec interrupt
QDEC_INTMask(QDEC, QDEC_Y_CT_INT_MASK, ENABLE);
//Read direction
dir = QDEC_GetAxisDirection(QDEC, QDEC_AXIS_Y);
}
}
