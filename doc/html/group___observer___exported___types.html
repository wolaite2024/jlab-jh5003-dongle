enum T_LE_SCAN_PARAM_TYPE

LE scan parameter type.

Enumerator
GAP_PARAM_SCAN_LOCAL_ADDR_TYPE 

The type of address being used in the scan request packets. Read/Write.

GAP_PARAM_SCAN_MODE 

Scan mode. Read/Write. Size is uint8. Default is GAP_SCAN_MODE_ACTIVE (T_GAP_SCAN_MODE).

GAP_PARAM_SCAN_INTERVAL 

Scan Interval. Read/Write. Size is uint16_t. Default is 0x10. Value range: 0x0004 - 0x4000 (2.5ms - 10240ms 0.625ms/step).

GAP_PARAM_SCAN_WINDOW 

Scan Window. Read/Write. Size is uint16_t. Default is 0x10. Value range: 0x0004 - 0x4000 (2.5ms - 10240ms 0.625ms/step).

GAP_PARAM_SCAN_FILTER_POLICY 

Scan Filter Policy. Read/Write. Size is uint8_t. Default is GAP_SCAN_FILTER_ANY (T_GAP_SCAN_FILTER_POLICY).

GAP_PARAM_SCAN_FILTER_DUPLICATES 

Scan Filter Duplicates. Read/Write. Size is uint8_t. Default is GAP_SCAN_FILTER_DUPLICATE_DISABLE (T_GAP_SCAN_FILTER_DUPLICATE).
