Detailed Description

USB isochronous packet information.

For isochronous transfer, multiple packets can be sent once, for example, if usb speed is high speed,
and bInterval of isochronous endpoint is 3, host will send TOKEN packet per 0.5ms, but upper application
may prepare data per 2 ms. In this case, upper application can process 4 packets once.

Parameters
offsetoffset of current packet in buf of T_HAL_USB_ISO_REQUEST_BLOCK
actualactual length of current packet
statusstatus of current packet transferred
