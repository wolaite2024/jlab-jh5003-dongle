LED_InitStruct Member Default Value

polarity LED_OUTPUT_NORMAL
prescale LED_TIME_TICK_SCALE
period_high[3] 0
period_low[3] 0
phase_update_rate[8] 0
phase_phase_tick[8] 0
phase_initial_duty[8] 0
phase_increase_duty[8] 0
phase_duty_step[8] 0
Parameters
[in]LED_InitStructPointer to a SLEEP_LED_InitTypeDef structure which will be initialized.

{
