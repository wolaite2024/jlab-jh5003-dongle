◆ ascs_app_ctl_handshake()

bool ascs_app_ctl_handshake ( uint16_t  conn_handle,
uint8_t  ase_id,
bool  app_handshake 
)

ASCS configures Application control start/stop ready operation.

ascs_mgr.h

This API is only used when the unicast server is in the audio sink role. This API is used to control whether the LE audio library sends the receiver stop ready operation or receiver start ready operation automatically.

Parameters
[in]conn_handleConnection handle.
[in]ase_idASE ID.
[in]app_handshakeApplication controls the handshake. The default value is false.
  • true Application controls the start/stop ready operation.
  • false Application does not control the start/stop ready operation.
Returns
The result of ASCS configuring Application control handshake.
Return values
trueASCS configuring Application control handshake is successful.
falseASCS configuring Application control handshake failed.
