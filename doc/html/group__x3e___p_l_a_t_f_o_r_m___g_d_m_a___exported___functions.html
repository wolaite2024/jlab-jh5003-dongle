bool(* GDMA_channel_request) (uint8_t *ch, void *isr, bool is_hp_ch)

Request an unused DMA channel.

Parameters
chRequested channel number. If there's no unused channel, ch is set to 0xa5.
isrRequested channel's isr. If isr is NULL, ignore ram vector table update.
is_hp_ch1: request high performance channel, 0: normal DMA channel.
Return values
trueStatus successful.
falseStatus fail.
