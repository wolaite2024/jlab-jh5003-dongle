◆ rscs_set_parameter()

bool rscs_set_parameter ( T_RSCS_PARAM_TYPE  param_type,
uint8_t  len,
void *  p_value 
)

Set a Running Speed and Cadence Service parameter.

This function can be called with a Running Speed and Cadence Service parameter type and it will set the Running Speed and Cadence Service parameter. Running Speed and Cadence Service parameters are defined in T_RSCS_PARAM_TYPE. If the "len" field is set to the size of a "uint16_t", the "p_value" field must point to data with the type of "uint16_t".

Parameters
[in]param_typeRunning Speed and Cadence Service parameter type: T_RSCS_PARAM_TYPE.
[in]lenLength of data to write.
[in]p_valuePointer to data to write. This is dependent on the parameter type and will be cast to the appropriate data type (for example: if data type of param is uint16_t, p_value will be cast to pointer of uint16_t).
Returns
Operation result.
Return values
trueOperation success.
falseOperation failure.

Example usage

void test(void)
{
bool op_result = rscs_set_parameter(RSCS_PARAM_TOTALDISTANCE, 4, (uint8_t*)&total_distance);
}
