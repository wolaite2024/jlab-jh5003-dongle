2 MPPGTool

MP Programming tool supports mass production and debug mode.

  1. The debug mode could only be enabled by input certain password, so that the developer could use the tool for debugging and feature development.
  2. The MP mode supports several features, such as programming up to 8 devices in parallel and changing BT address of device.

MP Pack, Flash Map Generator and OTA Header Generator functions are integrated into the MPPG tool chain. As for how to use these functions, please refer to its user guide in the tool\MPPG Tool directory.

Here, a brief overview of debug mode usage of the MP programming tool is provided. RTL87X3E is taken as an example to illustrate the steps of downloading. MPPG tool downloading connection may be different for RTL87X3D, please refer to the respective Evaluation Board User Guide for details.

Step 1: Connect USB connector CON3 on bottom side to PC as USB to UART input and supply power for the EVB, as introduced in 4.1.1 Power Supply in Quick Start User Guide .
Step 2: Make sure J33.2 FT_VIO is connected to J33.1 VIO1 (3.3V fixed, VDDIO0/LDO_HV33 in spec) to guarantee the voltage consistency.
Step 3: Use jumpers to connect J40.1 M3_0 to J41.4 FT232_TX and connect J40.2 M3_1 to J41.3 FT232_RX.
Step 4: Connect TP6 LOG to TP7 PD and then push RST key to reset SoC.

Figure 2-1 MPPG Tool Downloading Connection

Step 5: Open 'MPPGTool.exe' in /tool/MPPGTool-vX.X.X.X. Choose chip type and click the Confirm button.

Figure 2-2 RTL87X3E Chip Type

Step 6: Click 'Type' in the menu bar and choose 'Debug' mode, and the following interface will be displayed.

Figure 2-3 Choose Debug Mode

Step 7: Load flash map.ini in /bin/rtl87x3e/flash_map_config.

Figure 2-4 Load Flash Map

Step 8: Browse the folder and load the required bin file listed in 4.2 Download files in Quick Start User Guide in the following interface.

Figure 2-5 Load RTL87X3E Bin

Step 9: Click the Detect and Open button to open the port.

Figure 2-6 Open Port

Step 10: Click Chip Erase button to erase the chip before downloading. The status will be Success.

Figure 2-7 Chip Erase

Step 11: Click the Detect and Open button in tool UI again.
Step 12: Click the Download button and the status will be Success.

Figure 2-8 Download Bin

Step 13: Click Detect button to confirm whether the downloading progress is complete.
Step 14: Open the Factory Reset tool integrated into the MPPG tool chain

Figure 2-9 Open Factory Reset Tool

Step 15: Load the corresponding flash map ini and click Factory Reset Button

Figure 2-10 Factory Reset
Figure 2-11 Factory Reset Success

3 MCUConfigTool

MCU configuration tool is used for changing application and system behavior without any modification in source code.

  • 1. The system configuration is used to change system-wide configuration, such as logging, link numbers for different profile links, etc.
    • 1) stack options: BT stack and profile options
    • 2) platform options: system options, including internal charger
  • 2. The application configuration is designed only for audio application currently, and the purpose of this feature is to provide vendor simple way to change audio application UI parameters on its binary image.
    • 1) HW Feature
    • 2) Audio Route
    • 3) General
    • 4) System configuration
    • 5) BR/EDR
    • 6) LE
    • 7) Audio
    • 8) Charger
    • 9) Peripheral
    • 10) Ringtone
    • 11) Key
    • 12) LED
    • 13) RWS
    • 14) RF TX
    • 15) RHE
    • 16) Third Party

4 DebugAnalyzer

Debug analyzer is developed for generating system log via several transport layer and decoding it for user-friendly debugging.

Key features supported in this tool includes:

  • 1. Support multiple input ways
    • 1) UART
    • 2) Binary file
  • 2. Support both MCU and DSP logs
  • 3. Log saving and parsing
  • 4. Generate BT snoop log for further analyze by Frontline capture tool
  • 5. Support Ellisys injection

A brief introduction to the usage of Debug Analyzer is described as follows. Please refer to its user guide in the tool\Debug Anayzer directory for more details.

RTL87X3E is taken as an example to illustrate the steps. Log debugging connection may be different for RTL87X3D, please refer to the respective Evaluation Board User Guide for details.

Step 1: Firstly, please make sure pull off all the jumpers between J40 and J41.
Step 2: Connect J33.2 FT_VIO to J33.3 VIO2 (1.8V by default, VDDIO1/LDO_AUX in spec) to guarantee the voltage consistency.
Step 3: Connect TP6 LOG and J41.3 RX, and connect the USB connector CON3 on the bottom side to PC as USB to UART input.

Figure 4-1 Log Debugging Connection

Step 4: Open 'DebugAnalyzer.exe' in /tool/DebugAnalyzer-vX.X.X.X.
Step 5: Click the Settings option to set parameters.

Figure 4-2 Click Settings Option

Step 6: Set parameters in the following UI according to requirements and click Confirm button.

Figure 4-3 Settings Parameters

Step 7: Click Start button to print log.

Figure 4-4 Click Start Button

Step 8: Press the reset button in EVB, and log will be printed by Debug Analyzer as shown in the following figure.

Figure 4-5 Log Print

Step 9: User can switch the ARM log and DSP log by clicking the arrow in the lower left corner of UI.

Figure 4-6 ARM and DSP Log

Note: To generate system log, Log output should be set to Enable in System Configuration tab of McuConfigTool, as shown in the following figure.

5 Gadgets

Small tool sets for Keil application development are provided in SDK as well.

  1. flash: flash algorithm for Keil download option
  2. md5: generate md5 file format, which is necessary for Realtek MP tool
  3. prepend header: prepend header information with OTA and MP header for further MP process
  4. version generation: generate version for user application which may be useful for OTA and version track
  5. resign tool: re-sign all the images with the provided private key.

6 AudioConnect Android

Android tool set for BT audio could implement the features of TTS, OTA and renaming device. With the apk installed into android phone, the android tool set could communicate with BT audio devices and support the corresponding features described as below:

  1. OTA: over-the-air firmware update mechanism via BLE
  2. TTS: translate incoming caller information before answering the phone
  3. Rename: change the BT audio device name by phone

As for how to use the android tool set, please refer to its user guide in the tool\Android Tool Set directory.

7 AudioConnect iOS

iOS tool set is designed for iOS systems. It is quite similar to the android tool set, as it also support the features of TTS, OTA and renaming device.

As for how to use the iOS tool set, please refer to its user guide in the tool\iOS Tool Set directory.

8 DspConfigTool

The DSP configuration tool, allows developers to configure parameters of built in signal processing algorithms for phone call, music playback and VAD. In addition, this DSP configuration tool also provides user friendly parameter fine tuning functions, which can greatly shorten the turnaround time of parameter tuning. This tool also supports the generation of DSP configuration images.

As for how to use the DSP configuration tool, please refer to its user guide in the tool.

9 AncDesignTool

The AncDesignTool is convenient for users to choose the actual number of biquad filter used in a filter, design coefficients of biquad filters and visualize the overall response of a filter.

As for how to use the AncDesignTool, please refer to its user guide in the tool\AncDesignTool Set directory.

10 ANCMPTool

The ANCMPTool supports read gain, write gain and burn gain for feedforward and/or feedback filters. The write gain function can also be used to turn on or off feedforward and/or feedback filters.

As for how to use the ANCMPTool, please refer to its user guide in the tool.

11 SaiYanAncTool

The Saiyan tool is an acoustics MP calibration tool that supports component(speaker, mic, etc.) calibration and performance check. It ought to work with REALTEK SAIYAN board.

As for how to use the SaiYanAncTool, please refer to its user guide in the tool.

12 MPKitSetupPackage

The RF tool is used to verify RF performance and certification. This tool can generate non-singling Bluetooth signals of various modultion in regulatory certification and enter to direct test mode of Bluetooth.

As for how to use the MPKitSetupPackage, please refer to its user guide in the tool.

13 RealtekBTMPTool

BTMPTool is matched with Realtek's Genesis production version used in mass-production. The tool support write image, measure voltage, simple calibration frequency, audio loopback and modify config data in mass-production. The mp tool can support up to 1 to 8 mass production mode.

As for how to use the BTMPTool, please refer to its user guide in the tool.

14 FastMPTool

RtkFastMPTool must matched with Realtek's FastMp production Board used in mass-production. RtkFastMPTool allows the windows10 system of a computer to be simultaneously tested for functionally and the application is multithreaded. It can be allow to test 24 DUT include write image, RF calibration and modify config data at the same time.

As for how to use the FastMPTool, please refer to its user guide in the tool.

15 CFUDownload

CFUDownloadTool is used to upgrade firmware of the Realtek's chips that follow CFU (Component Firmware Update) protocol.

As for how to use the CFUDownloadTool, please refer to its user guide in the tool.

16 OtaDongleTool

The OTA Dongle Tool is used to programming OTA images in OTA tooling board for following spp OTA procedure for REALTEK Audio chip. The tool also supports controlling the OTA tooling board to do spp connection, force engage and spp OTA test.

As for how to use the OtaDongleTool, please refer to its user guide in the tool.

17 LocalPlayback Windows/Android/IOS

Localplayback tool includes Windows, Android and iOS version. Different system use different localplayback tool. The tool is convenient for users to manage music playlist and transfer music to evb SD flash.

  1. Localplayback tool (Windows): support edit playlist and transfer music to evb SD flash
  2. Localplayback tool (Android): single mode support edit playlist function and transfer music to evb SD flash, rws only support edit playlist function
  3. Localplayback tool (IOS): support edit playlist function

As for how to use the Localplayback tool, please refer to its user guide in the tool\Localplayback Tool Set directory.
