void UART_INTConfig ( UART_TypeDef *  UARTx,
uint32_t  UART_IT,
FunctionalState  newState 
)

Enables or disables the specified UART interrupts.

Parameters
UARTxselected UARTx peripheral.
UART_ITspecifies the UART interrupts sources to be enabled or disabled. This parameter can be any combination of the following values:
  • UART_INT_RD_AVA: enable Rx data avaliable interrupt.
  • UART_INT_FIFO_EMPTY: enable FIFO empty(TX FIFO empty) interrupt.
  • UART_INT_LINE_STS: enable line status interrupt.
  • UART_INT_MODEM_STS: enable modem status interrupt.
  • UART_INT_TX_DONE: enable tx done(TX FIFO empty and TX waveform sent done) interrupt
newStatenew state of the specified UART interrupts. This parameter can be: ENABLE or DISABLE.
Return values
None

  • UART_FLAG_RX_FIFO_ERR: rx FIFO error.
  • UART_FLAG_RX_IDLE: interrupt status of RX IDLE timeout.
  • UART_FLAG_TX_DONE: interrupt status of TX done(TX FIFO empty and TX waveform sent done).
Return values
Thenew state of UART_FLAG (SET or RESET).

void UART_MaskINTConfig ( UART_TypeDef *  UARTx,
uint32_t  UART_INT_MASK,
FunctionalState  NewState 
)

Mask or unmask the specified UART interrupts.

Added Since 2.13.0.0:
Parameters
UARTxselected UARTx peripheral.
UART_INT_MASKspecifies the UART interrupts sources to be masked or unmasked. This parameter can be any combination of the following values:
  • UART_INT_MASK_RD_AVA: mask Rx data avaliable interrupt.
  • UART_INT_MASK_TX_FIFO_EMPTY: mask TX FIFO empty interrupt.
  • UART_INT_MASK_RX_LINE_STS: mask line status interrupt.
  • UART_INT_MASK_RX_BREAK: mask RX break interrupt.
  • UART_INT_MASK_RX_IDLE: mask RX idle timeout interrupt.
  • UART_INT_MASK_TX_DONE: mask the interrupt of TXD done & TX_FIFO_EMPTY = 1.
  • UART_INT_MASK_TX_THD: mask TX fifo threshold interrupt.
NewStatenew state of the specified UART interrupts. This parameter can be: ENABLE or DISABLE.
