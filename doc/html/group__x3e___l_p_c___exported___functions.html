◆ LPC_SetTriggerEdge()

void LPC_SetTriggerEdge ( uint32_t  LPC_Edge)

Configure LPC trigger edge.

rtl876x_lpc.h

Added Since 2.13.0.0:
Parameters
[in]LPC_EdgeSpecifies the LPC trigger edge. This parameter can be one of the following values:
  • LPC_Vin_Below_Vth: The input voltage is below the threshold.
  • LPC_Vin_Over_Vth: The input voltage is above the threshold.
Returns
None.

Example usage

void driver_lpc_init(void)
{
LPC_SetTriggerEdge(LPC_Vin_Below_Vth);
}
