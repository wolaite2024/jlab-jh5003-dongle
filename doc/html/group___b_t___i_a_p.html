)

Send an iAP connection request.

bt_iap.h

Parameters
[in]bd_addrRemote BT address.
[in]server_channThe remote server_channel which can be found from the sdp info.
[in]frame_sizeThe max frame_size supported by local device.
[in]init_creditsThe number of packet that remote can be send. This para is used for flow control.A sending entity may send as many frames on an iAP connection as it has credits;if the credit count reaches zero,the sender will stop and wait for further credits from peer.
Returns
The status of sending the iAP connection request.
Return values
trueiAP connection request was sent successfully.
falseiAP connection request was failed to send.
