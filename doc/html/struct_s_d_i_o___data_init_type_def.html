Field Documentation

◆ SDIO_Address

uint32_t SDIO_Address

Specifies the start address of the Descriptor table. This parameter can be a value 4 aligned address.

◆ SDIO_BlockSize

uint32_t SDIO_BlockSize

Specifies the block size of data trnasfers for CMD17, CMD18, CMD24, CMD25 and CMD53, in the case of high capacity memory card, the size of data block is fixed to 512 bytes. This parameter can be a value of 0 to 0x800.

◆ SDIO_BlockCount

uint32_t SDIO_BlockCount

Specifies the block count for current transfer. This parameter can be a value of 0 to 0xffff.

◆ SDIO_TransferDir

uint32_t SDIO_TransferDir

Specifies the data transfer direction of DAT line data transfers. This parameter can be a value of IS_SDIO_TRANSFER_DIR.

◆ SDIO_TransferMode

uint32_t SDIO_TransferMode

Specifies the transfer mode.

◆ SDIO_DMAEn

uint32_t SDIO_DMAEn

Specifies whether to enable dma functionlity. This parameter can be a value of ENABLE or DISABLE.
