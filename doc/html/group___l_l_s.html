Detailed Description

Link loss service.

The Link Loss Service (LLS) defines behavior when a link is lost between two devices.

The Link Loss Service uses the Alert Level characteristic to cause an alert in the device when the link is lost.

Link Loss Service generally constitutes a profile collectively with other services, such as Proximity or Find Me etc., which enables the device to cause an alert when the link is lost.

Applications shall register link loss service during initialization through lls_add_service function.

Applications can set LLS alert level value through lls_set_parameter function.
