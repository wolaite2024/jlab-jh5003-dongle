enum T_GAP_LE_PARAM_TYPE

GAP LE Parameter Types List.

Enumerator
GAP_PARAM_DEV_STATE 

Device's current GAP device state. Read/Write. Size is sizeof(TGapDevState).

GAP_PARAM_APPEARANCE 

Local Device's Appearance. Read/Write. size is uint16. Appearance value please refer to GAP Appearance Values.(GAP_LE_APPEARANCE_VALUES).

GAP_PARAM_DEVICE_NAME 

Local Device's Name. Write Only. Name string length is GAP_DEVICE_NAME_LEN.

GAP_PARAM_SLAVE_INIT_GATT_MTU_REQ 

Slave initiates the GATT exchange MTU procedure. Write Only.

GAP_PARAM_RANDOM_ADDR 

Random address. Write Only.

GAP_PARAM_LATEST_CONN_BD_ADDR 

Latest connected Bluetooth device address. Read Only.

GAP_PARAM_LATEST_CONN_BD_ADDR_TYPE 

Latest connected Bluetooth device address type. Read Only.

GAP_PARAM_HANDLE_CREATE_CONN_IND 

APP handles the create connection indication message.

GAP_PARAM_DEFAULT_PHYS_PREFER 

Preferred values for the transmitter PHY and receiver PHY to be used for all subsequent connections over the LE transport.

GAP_PARAM_DEFAULT_TX_PHYS_PREFER 

The transmitter PHYs that the Host prefers the Controller to use.

GAP_PARAM_DEFAULT_RX_PHYS_PREFER 

The receiver PHYs that the Host prefers the Controller to use.

GAP_PARAM_USE_EXTENDED_ADV 

Use LE Advertising Extensions.

GAP_PARAM_DEFAULT_DATA_LEN_MAX_TX_OCTETS 

The Host's suggested value for the Controller's maximum transmitted number of payload octets in LL Data PDUs to be used for new connections. Range: 0x001B to 0x00FB.

GAP_PARAM_DEFAULT_DATA_LEN_MAX_TX_TIME 

The Host's suggested value for the Controller's maximum packet transmission time in microseconds for packets containing LL Data PDUs to be used for new connections. Range: 0x0148 to 0x4290.

GAP_PARAM_LOCAL_FEATURES 

Local supported features.

GAP_PARAM_DS_POOL_ID 

Downstream PoolID. Read only. size is uint16.

GAP_PARAM_DS_DATA_OFFSET 

Downstream pool buffer data offset. Read only. size is uint16.

GAP_PARAM_LE_REMAIN_CREDITS 

Remaining credits available for TX. Read only. size is uint16.

GAP_PARAM_MAX_WL_SIZE 

Max white list size. Read only. size is uint16.

GAP_PARAM_MIN_TX_POWER 

The minimum transmit powers supported by the Controller.

GAP_PARAM_MAX_TX_POWER 

The maximum transmit powers supported by the Controller.

GAP_PARAM_CIS_HOST_SUPPORT 

Host supports creating CISes.

GAP_PARAM_MASTER_INIT_GATT_MTU_REQ 

Master initiates the GATT exchange MTU procedure. Write Only. Size is uint8. Default value is true.
