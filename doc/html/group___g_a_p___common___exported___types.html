◆ T_GAP_PARAM_TYPE

enum T_GAP_PARAM_TYPE

GAP Parameter List.

Enumerator
GAP_PARAM_BD_ADDR 

Local Device's Address. Read Only. Size is uint8[GAP_BD_ADDR_LEN]. This item is read from the controller.

GAP_PARAM_BOND_PAIRING_MODE 

Pairing Mode of LE and BR/EDR. Write Only. Size is uint8_t. Default is GAP_PAIRING_MODE_PAIRABLE (Pairing Modes).

GAP_PARAM_BOND_AUTHEN_REQUIREMENTS_FLAGS 

Man-In-The-Middle (MITM) basically turns on Passkey protection in the pairing algorithm. Read/Write. Size is uint8_t. Default is GAP_AUTHEN_NO_MITM_YES_BOND (Mitm and Bonding Flags).

GAP_PARAM_BOND_IO_CAPABILITIES 

I/O capabilities. Read/Write. Size is uint8_t. Default is GAP_IO_CAP_NO_INPUT_NO_OUTPUT (T_GAP_IO_CAP).

GAP_PARAM_BOND_OOB_ENABLED 

OOB data available for pairing algorithm. Read/Write. Size is uint8_t. Default is 0(disabled).

GAP_PARAM_BOND_LINK_KEY_CONVERT 

Link Key Convert. Size is uint8_t.

GAP_PARAM_BOND_BR_PAIRING_MODE 

BR/EDR Pairing Mode. Read/Write. Size is uint8_t. Default is GAP_PAIRING_MODE_PAIRABLE (Pairing Modes).

GAP_PARAM_BOND_BR_AUTHEN_REQUIREMENTS_FLAGS 

BR/EDR Man-In-The-Middle (MITM) basically turns on Passkey protection in the pairing algorithm. Write. Size is uint8_t. Default is GAP_AUTHEN_NO_MITM_YES_BOND (Mitm and Bonding Flags).

GAP_PARAM_BOND_BR_IO_CAPABILITIES 

BR/EDR I/O capabilities. Write. Size is uint8_t. Default is GAP_IO_CAP_NO_INPUT_NO_OUTPUT (T_GAP_IO_CAP).

GAP_PARAM_BOND_LE_PAIRING_MODE 

LE Pairing Mode. Read/Write. Size is uint8_t. Default is GAP_PAIRING_MODE_PAIRABLE (Pairing Modes).

GAP_PARAM_BOND_LE_AUTHEN_REQUIREMENTS_FLAGS 

LE Man-In-The-Middle (MITM) basically turns on Passkey protection in the pairing algorithm. Write. Size is uint8_t. Default is GAP_AUTHEN_NO_MITM_YES_BOND (Mitm and Bonding Flags).

GAP_PARAM_BOND_LE_IO_CAPABILITIES 

LE I/O capabilities. Write. Size is uint8_t. Default is GAP_IO_CAP_NO_INPUT_NO_OUTPUT (T_GAP_IO_CAP).
