T_GAP_CAUSE le_ext_adv_set_adv_param ( uint8_t  adv_handle,

Set GAP extended advertising parameters for an advertising set.

Parameters
[in]adv_handleIdentify an advertising set, which is assigned by le_ext_adv_create_adv_handle.
[in]adv_event_propType of advertising event. Values for extended advertising PDUs: T_LE_EXT_ADV_EXTENDED_ADV_PROPERTY. Values for legacy advertising PDUs: T_LE_EXT_ADV_LEGACY_ADV_PROPERTY.
[in]primary_adv_interval_minMinimum advertising interval for undirected and low duty directed advertising. In units of 0.625ms, range: 0x000020 to 0xFFFFFF.
[in]primary_adv_interval_maxMaximum advertising interval for undirected and low duty directed advertising. In units of 0.625ms, range: 0x000020 to 0xFFFFFF.
[in]primary_adv_channel_mapA bit field that indicates the advertising channels that shall be used when transmitting advertising packets. Advertising Channel Map.
[in]own_address_typeLocal address type, T_GAP_LOCAL_ADDR_TYPE.
[in]peer_address_typeRemote address type, GAP_REMOTE_ADDR_LE_PUBLIC or GAP_REMOTE_ADDR_LE_RANDOM in T_GAP_REMOTE_ADDR_TYPE. GAP_REMOTE_ADDR_LE_PUBLIC: Public Device Address or Public Identity Address. GAP_REMOTE_ADDR_LE_RANDOM: Random Device Address or Random(static) Identity Address.
[in]p_peer_addressRemote address.
[in]filter_policyAdvertising filter policy: T_GAP_ADV_FILTER_POLICY.
[in]tx_powerAdvertising Tx power.
[in]primary_adv_phyIndicate the PHY on which the advertising packets are transmitted on the primary advertising channel. T_GAP_PHYS_PRIM_ADV_TYPE. If legacy advertising PDUs are being used, the primary_adv_phy shall indicate the LE 1M PHY (GAP_PHYS_PRIM_ADV_1M).
[in]secondary_adv_max_skipMaximum number of advertising events that can be skipped. Usually set to zero.
[in]secondary_adv_phyIndicate the PHY on which the advertising packets are transmitted on the secondary advertising channel. T_GAP_PHYS_TYPE.
[in]adv_sidSpecify the value to be transmitted in Advertising SID subfield of those advertising channel PDUs that have this field. Usually set to zero.
[in]scan_req_notification_enableIndicate whether Host will be notified upon the receipt of a scan request PDU. Usually set to false.
Returns
Operation result.
Return values
GAP_CAUSE_SUCCESSOperation success.
GAP_CAUSE_NOT_FINDOperation failure, the advertising handle is not found.

Example usage

T_APP_RESULT app_gap_callback(uint8_t cb_type, void *p_cb_data)
{
T_APP_RESULT result = APP_RESULT_SUCCESS;
T_LE_CB_DATA *p_data = (T_LE_CB_DATA *)p_cb_data;
switch (cb_type)
{
case GAP_MSG_LE_EXT_ADV_START_SETTING:
APP_PRINT_INFO3("GAP_MSG_LE_EXT_ADV_START_SETTING:cause 0x%x, flag 0x%x, adv_handle %d",
p_data->p_le_ext_adv_start_setting_rsp->cause, p_data->p_le_ext_adv_start_setting_rsp->flag,
p_data->p_le_ext_adv_start_setting_rsp->adv_handle);
if (p_data->p_le_ext_adv_start_setting_rsp->cause == GAP_CAUSE_SUCCESS)
{
// Initialize enable parameters
le_init_ext_adv_enable_params(p_data->p_le_ext_adv_start_setting_rsp->adv_handle);
// Enable one advertising set
le_ext_adv_enable(1, &p_data->p_le_ext_adv_start_setting_rsp->adv_handle);
}
break;
}
}

[in]durationIf non-zero, indicates the duration that the advertising set is enabled.
  • 0x0000: No advertising duration.
  • 0x0001-0xFFFF: Advertising duration, in units of 10ms.
[in]max_ext_adv_evtIf non-zero, indicates the maximum extended advertising events that shall be sent prior to disabling the extended advertising set.
  • 0x00: No maximum number of advertising events.
  • 0x01-0xFF: Maximum number of extended advertising events to send prior to terminating the extended advertising.
Returns
Operation result.
Return values
GAP_CAUSE_SUCCESSOperation success.
GAP_CAUSE_NOT_FINDOperation failure, the advertising handle is not found.

Example usage

void app_handle_ext_adv_state_evt(uint8_t adv_handle, T_GAP_EXT_ADV_STATE new_state, uint16_t cause)

ext_adv_state[i].adv_handle, ext_adv_state[i].ext_adv_state);

break;
}
}
APP_PRINT_INFO2("app_handle_ext_adv_state_evt: adv_handle %d newState %d",
adv_handle, new_state);
switch (new_state)
{
// device is idle
case EXT_ADV_STATE_IDLE:
{
APP_PRINT_INFO2("EXT_ADV_STATE_IDLE: adv_handle %d, cause 0x%x", adv_handle, cause);
}
break;
// device is advertising
case EXT_ADV_STATE_ADVERTISING:
{
APP_PRINT_INFO2("EXT_ADV_STATE_ADVERTISING: adv_handle %d, cause 0x%x", adv_handle, cause);
}
break;
default:
break;
}
}
