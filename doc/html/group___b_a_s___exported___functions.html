◆ bas_set_parameter()

bool bas_set_parameter ( T_BAS_PARAM_TYPE  param_type,
uint8_t  length,
uint8_t *  p_value 
)

Set a battery service parameter.

Call this function with a battery service parameter type and it will set the battery service parameter. Battery service parameters are defined in T_BAS_PARAM_TYPE. If the "len" field is set to the size of a "uint16_t", the "p_value" field must point to data with type of "uint16_t".

Parameters
[in]param_typeBattery service parameter type: T_BAS_PARAM_TYPE.
[in]lengthLength of data to write.
[in]p_valuePointer to data to write. This is dependent on the parameter type and will be cast to the appropriate data type. If param_type is set to BAS_PARAM_BATTERY_LEVEL, p_value points to the current charge level of a battery. Unit is org.bluetooth.unit.percentage. Minimum value is 0, and maximum value is 100. 100% represents fully charged while 0% represents fully discharged.
Returns
Operation result.
Return values
trueOperation success.
falseOperation failure.

Example usage

void test(void)
{
bool ret = bas_set_parameter(BAS_PARAM_BATTERY_LEVEL, 1, &battery_level);
}
