Introduction

The purpose of this document is to give an overview of the app sensor iqs773 module for customer. The document describes how to use sensor iqs773 module in app project. The source code consists of APP_SENSOR_IQS773 part. The details will be described in the following sections.

McuConfigTool Configuration

Psensor IQS773 can be enabled in "HW Feature" page .

Select psensor i2c and ready(Sensor interrupt) pin.

static void driver_init(void)
{
...

{
...
else if (app_cfg_const.psensor_vendor == PSENSOR_VENDOR_IQS773)
{
app_sensor_i2c_init(I2C_AZQ_ADDR, DEF_I2C_CLK_SPD, false);

{

else
