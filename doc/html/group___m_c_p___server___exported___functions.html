◆ mcp_server_read_cfm()

bool mcp_server_read_cfm ( T_MCP_SERVER_READ_CFM *  p_read_cfm)

Confirmation of read request sent by client.

This API should be used if all of the following conditions are true:

  • char_uuid of T_MCP_SERVER_READ_IND is MCS_UUID_CHAR_MEDIA_PLAYER_NAME , MCS_UUID_CHAR_TRACK_TITLE , MCS_UUID_CHAR_TRACK_DURATION , MCS_UUID_CHAR_TRACK_POSITION , or MCS_UUID_CHAR_CONTENT_CONTROL_ID
  • return value of callback registered by ble_audio_cback_register with msg LE_AUDIO_MSG_MCP_SERVER_READ_IND is BLE_AUDIO_CB_RESULT_PENDING

mcp_mgr.h

Experimental Added Since 2.13.0.0:
