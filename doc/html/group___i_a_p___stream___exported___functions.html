Process the event of the IAP.

Note
This API connects IAP stream module to IAP protocal by receiving and processing the IAP event. This API should be called where these events exist, with the required parameters, so that IAP stream can receive these events, and every event defined in T_IAP_STREAM_EVENT is requied
Parameters
[in]eventevent number
[in]event_paramevent parameter
Returns
void

Example Usage

// Examples of IAP connect events
// Suppose this location is where the IAP protocol is connected
T_IAP_STREAM_EVENT_PARAM iap_stream_param;
iap_stream_param.connect_param.mtu = mtu;
iap_stream_param.connect_param.remote_credits = credits;
iap_stream_param.connect_param.iap_index = iap_index;
iap_stream_event_process(IAP_STREAM_CONNECT_EVENT, &iap_stream_param);
