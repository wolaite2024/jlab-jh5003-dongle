bool tps_set_parameter ( T_TPS_PARAM_TYPE  param_type,
uint8_t  len,
void *  p_value 
)

Set a Tx power service parameter.

This function can be called with a tx power service parameter type and it will set the tx power service parameter. Tx power service parameters are defined in T_TPS_PARAM_TYPE. If the "len" field is set to the size of a "uint16_t", the "p_value" field must point to data with the type of "uint16_t".

Parameters
[in]param_typeTx power service parameter type: T_TPS_PARAM_TYPE.
[in]lenLength of data to write.
[in]p_valuePointer to data to write. This is dependent on the parameter type and WILL be cast to the appropriate data type (example: data type of uint16 will be cast to uint16 pointer).
Returns
Operation result.
Return values
trueOperation success.
falseOperation failure.

Example usage

void test(void)
{
bool ret = tps_set_parameter(TPS_PARAM_TX_POWER, 1, &tx_power);
}
