◆ WDG_Config()

void WDG_Config ( uint16_t  div_factor,
uint8_t  cnt_limit,
T_WDG_MODE  wdg_mode 
)

Config core watch dog.

Parameters
[in]div_factorThe div of watch dog, can be a value of 1 to 65535.
[in]cnt_limitThe cnt of watch dog. This parameter can be one of the following values:
  • 0: 0x001
  • 1: 0x003
  • 2: 0x007
  • 3: 0x00F
  • 4: 0x01F
  • 5: 0x03F
  • 6: 0x07F
  • 7: 0x0FF
  • 8: 0x1FF
  • 9: 0x3FF
  • 10: 0x7FF
  • 11~15: 0xFFF
[in]wdg_modeWatch dog mode T_WDG_MODE.

Example usage

void wdg_demo(void)
{
WDG_Config(1, 15, RESET_ALL);
}

Calculate div and cnt through timeout.

Parameters
[in]timeout_msTime to be calculated, can be a value of 0 to 8192000.
[out]div_factorThe div calculated by time.
[out]cnt_limitThe cnt_limit calculated by div and cnt. This parameter can be one of the following values:
  • 0: 0x001
  • 1: 0x003
  • 2: 0x007
  • 3: 0x00F
  • 4: 0x01F
  • 5: 0x03F
  • 6: 0x07F
  • 7: 0x0FF
  • 8: 0x1FF
  • 9: 0x3FF
  • 10: 0x7FF
  • 11~15: 0xFFF

Example usage

void wdg_demo(void)
{
uint16_t div_factor = 0;
uint8_t cnt_limit = 0;
WDG_Calculate_Timeout_ms(4000, &div_factor, &cnt_limit);
}
