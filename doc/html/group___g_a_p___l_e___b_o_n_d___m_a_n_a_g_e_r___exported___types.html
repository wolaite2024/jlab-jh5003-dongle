enum T_LE_BOND_PARAM_TYPE

LE bond parameter types.

Enumerator
GAP_PARAM_BOND_OOB_DATA 

OOB Data. Read/Write. size uint8_t[16]. Default is all 0's.

GAP_PARAM_BOND_FIXED_PASSKEY 

The fixed passcode for MITM protection. Read/Write. Size is uint32_t. Range is 0 - 999,999. Default is 0.

GAP_PARAM_BOND_FIXED_PASSKEY_ENABLE 

The fixed passcode available for pairing. Read/Write. Size is uint8_t. Default is 0 (disabled).

GAP_PARAM_BOND_SEC_REQ_ENABLE 

Send SMP security request when connected. Read/Write. Size is uint8_t. Default is 0 (disabled).

GAP_PARAM_BOND_SEC_REQ_REQUIREMENT 

Security request requirements. Read/Write. Size is uint8_t. Default is GAP_AUTHEN_BIT_BONDING_FLAG (Mitm and Bonding Flags).

GAP_PARAM_BOND_MIN_KEY_SIZE 

Minimum LTK size required. Read/Write. Size is uint8_t.

GAP_PARAM_BOND_GEN_LOCAL_IRK_AUTO 

Auto generate local IRK. Write only. Size is uint8_t. Default is 0 (disabled).

GAP_PARAM_BOND_SET_LOCAL_IRK 

Set local IRK. Write only. Size is uint8_t. Default is all zero.

GAP_PARAM_BOND_KEY_MANAGER 

Key manager. Write only. Size is uint8_t. Default is 0 (disabled).

GAP_PARAM_BOND_SIGN_KEY_FLAG 

Sign key flag configuration. Write only. Size is uint8_t. Default is 0 (disabled).

GAP_PARAM_BOND_CCCD_STORAGE 

Whether to store CCCD information. Write only. Size is uint8_t. Default is true.
