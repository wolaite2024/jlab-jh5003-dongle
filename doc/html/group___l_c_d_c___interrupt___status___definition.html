◆ LCDC_STATUS_WAVEFORM_FINISH_INT

#define LCDC_STATUS_WAVEFORM_FINISH_INT   BIT30

In DBI-B & SDI interface, the status will be set when CS signal return to inactive level. In eDPI interface, this status will be set when data transmission has been finished and stop state of PHY has been set.
