Field Documentation

◆ ScanClockKHZ

uint16_t ScanClockKHZ

Specifies the scan clock(KHz). Scan clock = source clock / (DIV+1), source clock is 20000KHz, DIV range is 0x0 to 0xFFF.

◆ DebonceClockKHZ

uint16_t DebonceClockKHZ

Specifies the debounce clock(KHz). Debounce clock = scan clock / (DEB_DIV+1), DEB_DIV range is 0x0 to 0xF.
