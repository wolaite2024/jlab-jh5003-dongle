  • APBPeriph_TIMERB_CLOCK

  • APBPeriph_SPI2W_CLOCK

  • APBPeriph_QDEC_CLOCK
  • APBPeriph_I2C2_CLOCK
  • APBPeriph_I2C1_CLOCK
  • APBPeriph_I2C0_CLOCK
  • APBPeriph_IR_CLOCK
  • APBPeriph_SPI2_CLOCK
  • APBPeriph_SPI1_CLOCK
  • APBPeriph_SPI0_CLOCK
  • APBPeriph_UART0_CLOCK
  • APBPeriph_UART1_CLOCK
  • APBPeriph_UART2_CLOCK
  • APBPeriph_UART3_CLOCK
  • APBPeriph_GPIOA_CLOCK
  • APBPeriph_GPIOB_CLOCK
  • APBPeriph_ADC_CLOCK
  • APBPeriph_I2S0_CLOCK
  • APBPeriph_I2S1_CLOCK
  • APBPeriph_CODEC_CLOCK
NewStatenew state of the specified peripheral clock. This parameter can be: ENABLE or DISABLE.
