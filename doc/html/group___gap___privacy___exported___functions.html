◆ le_privacy_get_param()

T_GAP_CAUSE le_privacy_get_param ( T_LE_PRIVACY_PARAM_TYPE  param,
void *  p_value 
)

Get an LE privacy parameter.

This function can be called with an LE Privacy Parameter type and it will get an LE Privacy Parameter. LE Privacy Parameters are defined in T_LE_PRIVACY_PARAM_TYPE.

Parameters
[in]paramGAP parameter type: T_LE_PRIVACY_PARAM_TYPE.
[in,out]p_valuePointer to location to get the parameter value. This is dependent on the parameter ID and will be cast to the appropriate data type (For example: if the data type of param is uint16_t, p_value will be cast to a pointer of uint16_t).
Returns
Operation result.
Return values
GAP_CAUSE_SUCCESSOperation success.
OthersOperation failure.
