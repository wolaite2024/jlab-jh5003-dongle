bool hids_set_parameter ( T_HIDS_PARAM_TYPE  param_type,
uint8_t  length,
void *  value_ptr 
)

Set a HID service parameter.

NOTE: You can call this function with a HID service parameter type and it will set the HID service parameter. HID service parameters are defined in T_HIDS_PARAM_TYPE.

Parameters
[in]param_typeHID service parameter type: T_HIDS_PARAM_TYPE
[in]lenLength of data to write
[in]p_valuePointer to data to write. This is dependent on the parameter type and WILL be cast to the appropriate data type
Returns
Operation result.
Return values
trueOperation success.
falseOperation failure.

Example usage

void test(void)
{
uint8_t mode = 1;
hids_set_parameter(HID_PROTOCOL_MODE, 1, &mode);
}
