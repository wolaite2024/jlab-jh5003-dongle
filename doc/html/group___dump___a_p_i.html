Detailed Description

raw data dump api

raw data dump, each has four trace types.

  1. ERROR : Fatal, Procedure Cannot Advance
  2. WARN : Abnormal Condition Occurred, But Procedure Can Advance
  3. INFO : Important Notification
  4. TRACE : Verbose Debug, Security Info
