Detailed Description

Use the combination of macro definitions to specify whether to only synchronize to periodic advertising with certain types of Constant Tone Extension (a value of 0 indicates that the presence or absence of a Constant Tone Extension is irrelevant).

Macro Definition Documentation

#define PA_SYNC_CREATE_SYNC_CTE_TYPE_CTE_IRRELEVANT   0x00

A value of 0 (i.e. all bits clear) indicates that the presence or absence of a Constant Tone Extension is irrelevant.
