◆ T_EVENT_GROUP

enum T_EVENT_GROUP

Event group definitions.

event code is 1 byte code to define messages type exchanging from/to different layers. first half byte is for group define, and second harf byte is for code define group code: 0x00: from stack layer 0x01: from peripheral layer 0x02: from framework 0x03: from app service 0x0A: from instructment

message group from BT layer

EVENT_GROUP_IO 

message group from IO layer

EVENT_GROUP_FRAMEWORK 

message group from Framework layer

EVENT_GROUP_APP 

message group from APP layer

EVENT_GROUP_INSTRUMENT 

message group from instructment layer
