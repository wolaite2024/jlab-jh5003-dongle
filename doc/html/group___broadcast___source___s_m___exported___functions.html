[in]primary_adv_interval_minMinimum advertising interval for undirected and low duty cycle directed advertising. Range: 0x000020 to 0xFFFFFF. Time = N * 0.625 ms, Time Range: 20 ms to 10,485.759375 s.
[in]primary_adv_interval_maxMaximum advertising interval for undirected and low duty cycle directed advertising. Range: 0x000020 to 0xFFFFFF Time = N * 0.625 ms, Time Range: 20 ms to 10,485.759375 s.
[in]primary_adv_channel_mapPrimary advertising channel map: Advertising Channel Map.
[in]own_address_typeOwn address type: T_GAP_LOCAL_ADDR_TYPE.
[in]p_local_rand_addrPointer to local random device address.
[in]filter_policyAdvertising filter policy: T_GAP_ADV_FILTER_POLICY.
[in]tx_powerAdvertising tx power. Range: -127 to +20, Units: dBm.
[in]primary_adv_phyPrimary advertising phy: T_GAP_PHYS_PRIM_ADV_TYPE.
[in]secondary_adv_max_skipMaximum advertising events the Controller can skip on the secondary advertising phy channel.
[in]secondary_adv_phySecondary advertising phy: T_GAP_PHYS_TYPE.
[in]broadcast_data_lenLength of broadcast data.
[in]p_broadcast_dataPointer to broadcast data.
Returns
The result of broadcast source set extended advertising parameters.
Return values

falseBroadcast source set extended advertising parameters failed.
