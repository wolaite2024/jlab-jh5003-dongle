◆ key_gpio_intr_callback()

void key_gpio_intr_callback ( uint32_t  key_index)

Key1~Key5 GPIO interrupt will be handle in this function. And they are all edge trigger. First disable app enter dlps and read current specific key GPIO input data bit. Disable key GPIO interrupt and send IO_MSG_TYPE_GPIO message to app task. Then enable key GPIO interrupt.
