◆ sps_set_parameter()

bool sps_set_parameter ( T_SPS_PARAM_TYPE  param_type,
uint8_t  len,
void *  p_value 
)

Set a scan paramter service parameter.

NOTE: You can call this function with a scan paramter service parameter type and it will set the scan paramter service parameter. Scan paramter service parameters are defined in T_SPS_PARAM_TYPE. If the "len" field sets to the size of a "uint16_t" ,the "p_value" field must point to a data with type of "uint16_t".

Parameters
[in]param_typeScan paramter service parameter type: T_SPS_PARAM_TYPE
[in]lenLength of data to write
[in]p_valuePointer to data to write. This is dependent on the parameter type and WILL be cast to the appropriate data type (For example: if data type of param is uint16_t, p_value will be cast to pointer of uint16_t).
Returns
Operation result.
Return values
trueOperation success.
falseOperation failure.

Example usage

void test(void)
{
uint8_t refresh_value = 10;
sps_set_parameter(SPS_PARAM_SCAN_REFRESH, 1, &refresh_value);
}
