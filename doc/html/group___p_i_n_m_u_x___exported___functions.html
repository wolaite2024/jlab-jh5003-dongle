rtl876x_pinmux.h

Parameters
[in]Pin_NumPin number to be configured. This parameter can be one of the following values:
  • P0_0 ~ P7_7, please refer to rtl876x.h 'Pin_Number' part.
[in]AON_PAD_ModeUse software mode or pinmux mode. This parameter can be one of the following values:
  • PAD_SW_MODE: Use software mode.
  • PAD_PINMUX_MODE: Use pinmux mode.
[in]AON_PAD_PwrOnConfig power of pad. This parameter can be one of the following values:
  • PAD_SHUTDOWN: Shutdown power of pad.
  • PAD_IS_PWRON: Enable power of pad.
[in]AON_PAD_PullConfig pad pull mode. This parameter can be one of the following values:
  • PAD_PULL_NONE: No pull.
  • PAD_PULL_UP: Pull this pin up.
  • PAD_PULL_DOWN: Pull this pin down.
[in]AON_PAD_EConfig pad output function, which only valid when PAD_SW_MODE. This parameter can be one of the following values:
  • PAD_OUT_DISABLE: Disable pin output.
  • PAD_OUT_ENABLE: Enable pad output.
[in]AON_PAD_OConfig pin output level, which only valid when PAD_SW_MODE and output mode. This parameter can be one of the following values:
  • PAD_OUT_LOW: Pad output low.
  • PAD_OUT_HIGH: Pad output high.
Returns
None.

Pad_Config(P2_1, PAD_PINMUX_MODE, PAD_IS_PWRON, PAD_PULL_UP, PAD_OUT_DISABLE, PAD_OUT_HIGH);
}
