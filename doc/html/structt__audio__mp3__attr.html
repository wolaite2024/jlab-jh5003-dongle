Detailed Description

Define Audio MP3 format attributes for AUDIO_FORMAT_TYPE_MP3.

Field Documentation

◆ sample_rate

uint32_t sample_rate

Sample rate for encoding or decoding. The supported values are 32000, 44100 and 48000 in HZ for MPEG-1 Layer 3; Additional 16000, 22050 and 24000 in HZ for MPEG-2 Layer 3, and additional 8000, 11025 and 12000 in HZ for MPEG-2.5 Layer 3.

◆ chann_mode

T_AUDIO_MP3_CHANNEL_MODE chann_mode

Channel mode for encoding or decoding. The supported values are defined in T_AUDIO_MP3_CHANNEL_MODE.

◆ version

uint8_t version

MPEG Audio version ID for encoding or decoding. The supported values are 0 for MPEG Version 2.5 (unofficial), 1 for reserved, 2 for MPEG Version 2 (ISO/IEC 13818-3), and 3 for MPEG Version 1 (ISO/IEC 11172-3).

◆ layer

uint8_t layer

MPEG Audio Layer description for encoding or decoding. The supported values are 0 for reserved, 1 for Layer 3, 2 for Layer 2, and 3 for Layer 1.

◆ bitrate

uint32_t bitrate

Bitrate for encoding or decoding. The supported ranges are from 8000 bit/s to 448000 bit/s.
