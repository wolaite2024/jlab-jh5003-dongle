void Pinmux_Config ( uint8_t  Pin_Num,
uint8_t  Pin_Func 
)

Config the selected pin to its corresponding IO function.

rtl876x_pinmux.h

Parameters
[in]Pin_NumPin number to be configured. This parameter can be one of the following values:
  • P0_0 ~ P10_0, please refer to rtl876x.h "Pin_Number" part.
[in]Pin_Funcmean one IO function, can be a value of Pin Function Number.
Returns
None.

Example usage

void driver_uart_init(void)
{
Pad_Config(P2_0, PAD_PINMUX_MODE, PAD_IS_PWRON, PAD_PULL_UP, PAD_OUT_DISABLE,
PAD_OUT_HIGH);
Pad_Config(P2_1, PAD_PINMUX_MODE, PAD_IS_PWRON, PAD_PULL_UP, PAD_OUT_DISABLE,
PAD_OUT_HIGH);
Pinmux_Config(P2_0, UART0_TX);
Pinmux_Config(P2_1, UART0_RX);
}

Config the selected pin to its corresponding IO function.

Pin_Numpin number. This parameter is from ADC_0 to P4_1, please refer to rtl876x.h "Pin_Number" part.

Return values
