bool hal_gpio_register_isr_callback ( uint8_t  pin_index,
P_GPIO_CBACK  callback,
uint32_t  context 
)

Register the GPIO interrupt callback for the specific pin.

hal_gpio_int.h

Parameters
[in]pin_indexThe pin index, please refer to rtl876x.h 'Pin_Number' part.
[in]callbackThe callback to be called when the specific interrupt happened.
[in]contextThe user data when callback is called.
Returns
The status of register interrupt functions.
Return values
trueThe callback is registered successfully.
falseThe callback is failed to register due to invalid pin number.

Example usage

void gpio_test(void)
{
hal_gpio_set_up_irq(TEST_PIN, GPIO_IRQ_EDGE, GPIO_IRQ_ACTIVE_LOW, true);
hal_gpio_register_isr_callback(TEST_PIN, gpio_isr_cb, GPIO_DEMO_INPUT_PIN0);
hal_gpio_irq_enable(TEST_PIN);
}
