◆ DBIB_BypassCmdByteCmd()

void DBIB_BypassCmdByteCmd ( FunctionalState  NewState)

Enable or disable command byte bypass in auto mode.

Parameters
[in]NewStateNew state of command byte bypass. This parameter can be one of the following values:
  • ENABLE: Enable command byte bypass.
  • DISABLE: Disable command byte bypass.

Example usage

void driver_lcdc_dbib_init(void)
{
DBIB_BypassCmdByteCmd(ENABLE);
}

◆ DBIB_Init()

void DBIB_Init ( LCDC_DBIBCfgTypeDef *  DBIBCfg)

Initializes DBIB interface according to the specified parameters in the DBIBCfg.

Parameters
[in]DBIBCfgPointer to a LCDC_DBIBCfgTypeDef structure that contains the configuration information for DBIB interface.

Example usage

void driver_lcdc_dbib_init(void)
{
LCDC_DBIBCfgTypeDef dbib_init = {0};
dbib_init.DBIB_SPEED_SEL = DBIB_CLOCK_DIV_2;
dbib_init.DBIB_InitGuardTimeCmd = DBIB_INIT_GUARD_TIME_DISABLE;
dbib_init.DBIB_InitGuardTime = DBIB_INIT_GUARD_TIME_1T;
dbib_init.DBIB_CmdGuardTimeCmd = DBIB_CMD_GUARD_TIME_DISABLE;
dbib_init.DBIB_CmdGuardTime = DBIB_CMD_GUARD_TIME_1T;
dbib_init.DBIB_GuardTimeCmd = DBIB_GUARD_TIME_DISABLE;
dbib_init.DBIB_GuardTime = DBIB_GUARD_TIME_1T;
dbib_init.DBIB_WRDelay = DBIB_WR_HALF_DELAY;
DBIB_Init(&dbib_init);
}

void DBIB_StructInit ( LCDC_DBIBCfgTypeDef *  DBIBCfg)

Initialize the LCDC_DBIBCfgTypeDef structure with default values.

Parameters

Note
Default values for each member:
Member Default Value

DBIB_InitGuardTimeCmd DBIB_INIT_GUARD_TIME_DISABLE
DBIB_InitGuardTime DBIB_GUARD_TIME_1T
DBIB_CmdGuardTimeCmd DBIB_CMD_GUARD_TIME_DISABLE
DBIB_CmdGuardTime DBIB_GUARD_TIME_1T
