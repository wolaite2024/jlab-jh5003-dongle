bool ccp_server_read_cfm ( T_CCP_SERVER_READ_CFM *  p_read_cfm)

Confirmation of read request sent by client.

This API should be used if all of the following conditions are true:

  • char_uuid of T_CCP_SERVER_READ_IND is TBS_UUID_CHAR_BEARER_PROVIDER_NAME , TBS_UUID_CHAR_BEARER_TECHNOLOGY , TBS_UUID_CHAR_BEARER_URI_SCHEMES_SUPPORTED_LIST , TBS_UUID_CHAR_BEARER_SIGNAL_STRENGTH , TBS_UUID_CHAR_STATUS_FLAGS , TBS_UUID_CHAR_BEARER_UCI , TBS_UUID_CHAR_BEARER_SIGNAL_STRENGTH_REPORTING_INTERVAL , or TBS_UUID_CHAR_CONTENT_CONTROL_ID
  • return value of callback registered by ble_audio_cback_register with msg LE_AUDIO_MSG_CCP_SERVER_READ_IND is BLE_AUDIO_CB_RESULT_PENDING

ccp_mgr.h
