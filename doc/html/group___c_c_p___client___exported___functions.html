Read characteristic value.

If sending operation is successful, the result of read characteristic value will be returned by callback registered by ble_audio_cback_register with msg LE_AUDIO_MSG_CCP_CLIENT_READ_RESULT and buf T_CCP_CLIENT_READ_RESULT.

ccp_client.h

Parameters
[in]conn_handleConnection handle.
[in]srv_instance_idService instance ID.
[in]char_uuid
  • TBS_UUID_CHAR_BEARER_PROVIDER_NAME
  • TBS_UUID_CHAR_BEARER_UCI
  • TBS_UUID_CHAR_BEARER_TECHNOLOGY
  • TBS_UUID_CHAR_BEARER_URI_SCHEMES_SUPPORTED_LIST
  • TBS_UUID_CHAR_BEARER_SIGNAL_STRENGTH
  • TBS_UUID_CHAR_BEARER_SIGNAL_STRENGTH_REPORTING_INTERVAL
  • TBS_UUID_CHAR_BEARER_LIST_CURRENT_CALLS
  • TBS_UUID_CHAR_STATUS_FLAGS
  • TBS_UUID_CHAR_INCOMING_CALL_TARGET_BEARER_URI
  • TBS_UUID_CHAR_CALL_STATE
  • TBS_UUID_CHAR_CALL_CONTROL_POINT_OPTIONAL_OPCODES
  • TBS_UUID_CHAR_INCOMING_CALL
  • TBS_UUID_CHAR_CALL_FRIENDLY_NAME
  • TBS_UUID_CHAR_CONTENT_CONTROL_ID
[in]gtbs
  • true : Generic Telephone Bearer Service (GTBS).
  • false : Telephone Bearer Service (TBS).
Returns
The result of the sending operation.
Return values
trueSending operation is successful.
falseSending operation is failed.

Write Call Control Point characteristic.

If sending operation is successful and Call Control Server sends Call Control Point Notification, Call Control Point Notification will be returned by callback registered by ble_audio_cback_register with msg LE_AUDIO_MSG_CCP_CLIENT_CALL_CP_NOTIFY and buf T_CCP_CLIENT_CALL_CP_NOTIFY.

ccp_client.h

Parameters
[in]conn_handleConnection handle.
[in]srv_instance_idService instance ID.
[in]gtbs
  • true : Generic Telephone Bearer Service (GTBS).
  • false : Telephone Bearer Service (TBS).
[in]is_req
  • true : Write request.
  • false : Write command.
[in]p_paramPointer to call control point parameter: T_CCP_CLIENT_WRITE_CALL_CP_PARAM.
Returns
The result of the sending operation.
Return values
trueSending operation is successful.
falseSending operation is failed.

Example usage

void test(void)
{
bool retval = ccp_client_write_call_cp(conn_handle, srv_instance_id, gtbs, is_req, p_param);
}
