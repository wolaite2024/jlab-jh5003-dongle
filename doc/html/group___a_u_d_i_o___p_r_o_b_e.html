Detailed Description

Test, analyze and make profiling on the Audio Subsystem.

Audio Probe provides the testing or profiling interfaces, which application can analyze the Audio Subsystem.

Note
Audio Probe interfaces are used only for testing and profiling. Application shall not use Audio Probe interfaces in any formal product scenarios.
