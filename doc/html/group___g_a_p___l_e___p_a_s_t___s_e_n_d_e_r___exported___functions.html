T_GAP_CAUSE le_past_sender_periodic_adv_set_info_transfer ( uint8_t  conn_id,
uint16_t  service_data,
uint8_t  adv_handle 
)

Request the Controller to send synchronization information about the periodic advertising in an advertising set to a connected device.

If sending request operation is successful, the result will be returned by app_gap_callback with cb_type GAP_MSG_LE_PAST_SENDER_PERIODIC_ADV_SET_INFO_TRANSFER.

Parameters
[in]conn_idUsed to identify a connection.
[in]service_dataA value provided by the Host to identify the periodic advertising train to the peer device.
[in]adv_handleUsed to identify the advertising set.
Returns
Send request operation.
Return values
GAP_CAUSE_SUCCESSSend request operation success.
OthersSend request operation failure.

Example usage

{
T_GAP_CAUSE cause = le_past_sender_periodic_adv_set_info_transfer(conn_id, service_data, adv_handle);
}
