pasc_mgr.h

Parameters
[in]directionAudio direction: T_AUDIO_DIRECTION.
[in]pac_dataPointer to the PAC data. This parameter shall not use local variables, and audio lib only holds Pointers internally.
[in]pac_data_lenPAC data length. This parameter shall not use local variables.
[in]notifyWhether notify is supported.

  • false Notify is not supported.

PAC id.
