◆ SPI_BaudRatePrescaler

uint32_t SPI_BaudRatePrescaler

Specifies the speed of SCK clock. SPI Clock Speed = clk source/SPI_ClkDIV

Note
The communication clock is derived from the master clock. The slave clock does not need to be set.

Specifies the speed of SCK clock. SPI Clock Speed = clk source(fixed 40Mhz)/SPI_ClkDIV/SPI_BaudRatePrescaler, clk source is 40Mhz by default, SPI_ClkDIV refer API RCC_SPIClkDivConfig SPI_BaudRatePrescaler: should be any even value between 2 and 65534

Note
The communication clock is derived from the master clock. The slave clock does not need to be set.
