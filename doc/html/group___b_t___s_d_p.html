◆ bt_sdp_discov_start()

bool bt_sdp_discov_start ( uint8_t  bd_addr[6],
T_BT_SDP_UUID_TYPE  uuid_type,
T_BT_SDP_UUID_DATA  uuid_data 
)

Start SDP discovery.

Parameters
[in]bd_addrBluetooth address of remote device.
[in]uuid_typeType of UUID to discovery.
  • BT_SDP_UUID16 UUID in 16 bits.
  • BT_SDP_UUID32 UUID in 32 bits.
  • BT_SDP_UUID128 UUID in 128 bits.
[in]uuid_dataDiscovery SDP UUID data corresponding to uuid_type.
Returns
The status of starting SDP discovery.
Return values
trueSDP discovery was started successfully.
falseSDP discovery was failed to start.
