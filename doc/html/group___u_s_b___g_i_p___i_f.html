struct  _usb_gip_pipes
 USB GIP PIPE callbacks to transmit audio data or control infomation to upper sw
ctrl: transmit vol and mute control infomation, param pipe is defined
in T_USB_GIP_PIPES, event is defined in T_USB_GIP_CTRL_EVT,
ctrl is defined in T_USB_GIP_CTRL_ATTR. upstream: transmit data from device to host, param pipe is defined in
T_USB_GIP_PIPES, buf is audio data will be sent, len is length of
buf
downstream: transmit data from host to device, param pipe is defined in
T_USB_GIP_PIPES, buf is audio data has been received, len is length
of buf
