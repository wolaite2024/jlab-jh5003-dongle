SPDIF clock source config.

Parameters
ClockSourcespecifies the SPDIF clock sorce. this parameter can be one of the following values:
  • SPDIF_CLOCK_SOURCE_40M
  • SPDIF_CLOCK_SOURCE_PLL
  • SPDIF_CLOCK_SOURCE_PLL2
  • SPDIF_CLOCK_SOURCE_PLL3
  • SPDIF_CLOCK_SOURCE_MCLK
first_divspecifies the SPDIF clock souce first div, actual effective division value = first_div + 1.
sec_divspecifies the SPDIF clock souce second div. this parameter can be one of the following values:
  • SPDIF_CLOCK_SEC_DIV_DISABLE
  • SPDIF_CLOCK_SEC_DIV_2
Return values
None
