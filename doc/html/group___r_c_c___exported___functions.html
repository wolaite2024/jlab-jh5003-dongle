void RCC_PeriphClockCmd ( uint32_t  APBPeriph,
uint32_t  APBPeriph_Clock,
FunctionalState  NewState 
)

Enable or disable the APB peripheral clock.

rtl876x_rcc.h

Parameters
[in]APBPeriphSpecifies the APB peripheral to gates its clock. This parameter can be one of the following values:
  • APBPeriph_TIMERx: The APB peripheral of TIMER, where x can be A or B.
  • APBPeriph_GDMA: The APB peripheral of GDMA.
  • APBPeriph_SPI2W: The APB peripheral of SPI2W.
  • APBPeriph_KEYSCAN: The APB peripheral of KEYSCAN.
  • APBPeriph_QDEC: The APB peripheral of QDEC.
  • APBPeriph_I2Cx: The APB peripheral of I2C, where x can be 0 to 2.
  • APBPeriph_IR: The APB peripheral of IR.
  • APBPeriph_SPIx: The APB peripheral of SPI, where x can be 0 to 2.
  • APBPeriph_GPIOx: The APB peripheral of GPIO, where x can be A to C.
  • APBPeriph_UARTx: The APB peripheral of UART, where x can be 0 to 3.
  • APBPeriph_ADC: The APB peripheral of ADC.
  • APBPeriph_CODEC: The APB peripheral of CODEC.
[in]APBPeriph_ClockSpecifies the APB peripheral clock config. This parameter can be one of the following values(must be the same with APBPeriph):
  • APBPeriph_TIMERx_CLOCK: The APB peripheral clock of TIMER, where x can be A or B.
  • APBPeriph_GDMA_CLOCK: The APB peripheral clock of GDMA.
  • APBPeriph_SPI2W_CLOCK: The APB peripheral clock of SPI2W.
  • APBPeriph_KEYSCAN_CLOCK: The APB peripheral clock of KEYSCAN.
  • APBPeriph_QDEC_CLOCK: The APB peripheral clock of QDEC.
  • APBPeriph_I2Cx_CLOCK: The APB peripheral clock of I2C, where x can be 0 to 2.
  • APBPeriph_IR_CLOCK: The APB peripheral clock of IR.
  • APBPeriph_SPIx_CLOCK: The APB peripheral clock of SPI, where x can be 0 to 2.
  • APBPeriph_GPIOx_CLOCK: The APB peripheral clock of GPIO, where x can be A to C.
  • APBPeriph_UARTx_CLOCK: The APB peripheral clock of UART, where x can be 0 to 3.
  • APBPeriph_ADC_CLOCK: The APB peripheral clock of ADC.
  • APBPeriph_CODEC_CLOCK: The APB peripheral clock of CODEC.
[in]NewStateNew state of the specified peripheral clock. This parameter can be one of the following values:
  • ENABLE: Enable the specified peripheral clock.
  • DISABLE: Disable the specified peripheral clock.
Returns
None.

Example usage

void driver_spi_init(void)
{
RCC_PeriphClockCmd(APBPeriph_SPI0, APBPeriph_SPI0_CLOCK, ENABLE);
}

None.

◆ RCC_SPDIFClkSourceConfig()

void RCC_SPDIFClkSourceConfig ( uint16_t  ClockSource,
uint8_t  first_div,
uint8_t  sec_div 
)

SPDIF clock source config.

rtl876x_rcc.h

Parameters
[in]ClockSourceSpecifies the SPDIF clock sorce. This parameter can be one of the following values:
  • SPDIF_CLOCK_SOURCE_40M: Select SPDIF clock source of 40MHz.
  • SPDIF_CLOCK_SOURCE_PLL: Select SPDIF clock source of PLL.
  • SPDIF_CLOCK_SOURCE_PLL2: Select SPDIF clock source of PLL2.
  • SPDIF_CLOCK_SOURCE_PLL3: Select SPDIF clock source of PLL3.
  • SPDIF_CLOCK_SOURCE_MCLK: Select SPDIF clock source of MCLK.
[in]first_divSpecifies the SPDIF clock source first div, actual effective division value = first_div + 1.
[in]sec_divSpecifies the SPDIF clock source second div. This parameter can be one of the following values:
  • SPDIF_CLOCK_SEC_DIV_DISABLE: Disable the SPDIF clock source second divider.
  • SPDIF_CLOCK_SEC_DIV_2: The SPDIF clock source second divider is 2.
Returns
None.

Example usage

void driver_spdif_init(void)
{
RCC_SPDIFClkSourceConfig(SPDIF_CLOCK_SOURCE_40M, 1, SPDIF_CLOCK_SEC_DIV_2);
}
